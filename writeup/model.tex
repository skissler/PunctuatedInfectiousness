\subsection{The time-shifted Gamma model}

\begin{itemize}
    \item While we seek a general understanding of how $a_i(\tau)$, independent of $A(\tau)$, impacts epidemic dynamics, we here introduce a particular modeling framework --- that is nevertheless quite general --- that allows us to be explicit about our claims. 
    \item Specifically, we want a way of defining $a_i(\tau)$ so that: 
    \begin{itemize}
        \item $A(\tau) = E_i[a_i(\tau)] = \lim_{n \rightarrow \infty }\frac{1}{n} \sum_{i=1}^n a_i(\tau)$; required be definition.
        \item $\nu_i  = R_0 \, \forall i$; that is, there is no inherent interpresonal variation in the level of infectiousness (no baked-in overdispersion). 
        \item The width of $a_i(\tau)$ is governed by a single smoothness parameter $\psi \in [0,1]$, where $\psi \rightarrow 1$ makes $a_i(\tau) \equiv A(\tau)$ (maximally smooth; each person's individual infectiousness profile follows the population average) and $\psi \rightarrow 0$ makes $a_i(\tau)$ a delta function (maximally narrow; each person's secondary infections all occur at the same time). 
    \end{itemize}
    \item We begin by assuming the intrinsic generation interval distribution is Gamma-distributed, which is a common assumption in applied epidemiology. Specifically, $g(\tau)$ follows a $\text{Gamma}(\alpha, \beta)$ distribution where $\beta = \alpha / \bar{g}$ and $\bar{g}$ is the mean generation time. Thus, $A(\tau) = R_0\,g(\tau)$ follows a rescaled Gamma distribution. 
    \item Now, define 
    \begin{equation}
        \xi_i(\tau) = \begin{cases}
          0 & \tau \leq l_i \\
          f_\epsilon(\tau - l_i) & \tau > l_i
        \end{cases}
    \end{equation}
    where $l_i \sim \text{Gamma}((1 - \psi)\alpha, \beta)$ and $f_\epsilon(\tau)$ is a $\text{Gamma}(\psi \alpha, \beta)$ pdf. Furthermore, let $a_i(\tau) = R_0 \xi_i(\tau)$. This satisfies the three requirements: 
    \begin{itemize}
        \item Expectation argument, needs to be made explicit 
        \item $\nu_i = R_0$ by construction
        \item $\psi \rightarrow 1$ makes $a_i(\tau) \equiv A(\tau)$, with distribution $\text{Gamma}(\alpha, \beta)$. $\psi \rightarrow 0$ yields $f_\epsilon(\tau)$ following a Gamma distribution with shape parameter approaching 0 from above, which has 0 variance, \ie, a delta function. 
    \end{itemize}
    \item Simulations confirm that this works (Figure). 
\end{itemize}

\subsection{Allowing for time-varying contacts}

\begin{itemize}
    \item The basic formulation assumes interpersonal contacts are time-homogeneous; this is what allows $A(\tau)$ and $a_i(\tau)$ to be functions of time-since-infection $\tau$ but not calendar time $t$. We will follow this convention for most of our analyses. 
    \item However, there are important instances where we want to consider how punctuated infectiousness interacts with time-varying contacts. 
    \item To achieve this, we introduce a generalization on the model and two specific models of interpersonal contacts that we will use as testbeds. 
    \item We define the time-homogeneous contact process $\bar{c} \equiv 1$, and we can explicitly introduce this contact process into the expression for $a_i(\tau)$: 
    \begin{equation}
        a_i(\tau) = R_0 \xi_i(\tau) \bar{c}
    \end{equation}
    \item We now introduce $c(t)$, which captures proportional deviations from $\bar{c}$. We require $E_t[c(t)] = 1$, that is, at a random time, the expected value is 1; this is achieved by periodic functions with mean value 1 and by random processes with mean value 1. Thus: 
    \begin{equation}
        a_i(\tau) = R_0 \xi_i(\tau) c(t)
    \end{equation}
    and now $R_0$ and $\xi_i(\tau)$ are specifically the expected number of secondary infections and the expected timing of infections under the reference contact process $\bar{c}$. If $c(t)$ varies in time, we get 
    \begin{itemize}
        \item $\nu_i^\text{rel} = $ (expression for realized basic reproduction number)
        \item $xi_i^\text{rel}(t, \tau) = $ (expression for realized generation interval distribution)
    \end{itemize}
    \item Thus $a_i(\tau, t) = \nu_i \xi_i(\tau) c(t+\tau) = \nu_i^\text{rel} \xi_i^\text{rel}(t, \tau)$
    \item As test cases, we consider:

    \end{itemize}
\end{itemize}

\subsection{Simulation approach}

We investigate the impact of punctuatedness on epidemic dynamics using both closed-form analytic results and stochastic simulations. Stochastic simulations use a branching-process-like algorithm in a finite population of $N = 10{,}000$ susceptible individuals with a single initial infectious individual. For each infected person $i$:
%
\begin{enumerate}
    \item An inherent generation interval distribution $\xi_i(\tau)$ is drawn from the time-shifted Gamma model (Eq.~\ref{eq:gamma_timeshift}). 
    \item The number of secondary infection attempts is drawn as $X_i \sim \text{Poisson}(R_0)$.
    \item The timing of each attempt relative to infection is drawn from $\xi_i(\tau)$.
    \item When the contact process is non-uniform ($c_i \neq 1$), Poisson thinning is applied: each attempt at calendar time $t_i + \tau$ is accepted with probability proportional to $c_i(t_i + \tau)$, ensuring that the time-dependent infection rate matches $a_i(t_i, \tau)$.
    \item Each accepted attempt is paired with a uniformly chosen individual from the population: if the recipient is susceptible, infection occurs; otherwise, the attempt fails.
\end{enumerate}
%
Full details are provided in the {\bf Supplementary Methods}. For illustration, we focus on three sets of epidemiological parameters roughly pertaining to influenza-like, SARS-CoV-2 omicron-like, and measles-like pathogens ({\bf Table XX}).

% To investigate the impact of the punctuatedness of $a_i(\tau)$ on epidemic dynamics, we turn to both analytic results and stochastic simultions. For each person $i$, a biological and contact infectiousness potential ($b_i$ and $c_i$), and thus the full individual infectiousness profile ($a_i$), are generated {\it a priori}. Infection attempts are generated for each person by drawing a number of infections $X \sim \text{Poisson}(\nu_i)$, where $\nu_i = \int_0^\infty a_i(\tau) d\tau$. These infections are then distributed according to $a_i(\tau)$ using Gamma-distributed draws. Infection attempts are paired with ``recipients'' uniform-randomly: if the recipient is susceptible, infection succeeds, while if the person has already been infected, the infection attempt fails.
