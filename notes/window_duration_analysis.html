Window Duration Analysis for Growth Rate Estimation

Analysis Date: 2026-04-16

Parameters: N = 10,000 | Z_min = 100

Executive Summary

Critical Finding: It is impossible to satisfy both constraints simultaneously with a single window duration D for all three pathogens.
  • To achieve ≥2 generation times: D ≥ 24.4 days
  • To maintain <5% susceptible depletion: D < 7.1 days
This incompatibility arises because Measles has a very long generation time (12.2 days) while simultaneously having a slow growth rate (r=0.228 per day).

Constraint Definitions and Analysis

Constraint 1: Statistical Power

Requirement: Window should span ≥2 generation times for all pathogens, ensuring sufficient time for reliable growth rate estimation.

Mathematical formulation:

D ≥ 2 × max(T_gen) = 2 × 12.2 = 24.4 days

Implication: No window smaller than ~24 days will satisfy this constraint for all pathogens.

Constraint 2: Minimal Susceptible Depletion

Requirement: Susceptible pool depletion should remain <5% to preserve the validity of the exponential growth model (which assumes unlimited susceptibles).

Mathematical formulation:

Z/N < 0.05 ⟹ Z < 500 ⟹ 100 × exp(r×D) < 500 exp(r×D) < 5 ⟹ D < ln(5)/r

For each pathogen:

  • Influenza (r=0.252): D < ln(5)/0.252 = 6.39 days
  • Omicron (r=0.378): D < ln(5)/0.378 = 4.26 days
  • Measles (r=0.228): D < ln(5)/0.228 = 7.06 days ← Most restrictive

Implication: To satisfy this constraint for all pathogens, D < 4.26 days (Omicron's limit).

The Incompatibility

Constraint Requirement
Generation times (≥2) D ≥ 24.4 days
Depletion (<5%) D < 4.26 days
OVERLAP: None. Ranges are disjoint.

Critical Caveat: Model Validity

The Exponential Model Breaks When Z Becomes Large

The exponential growth model Z(t) = Z_min × exp(r×t) assumes an unlimited susceptible pool. Once cumulative incidence approaches a material fraction of N, three effects invalidate this model:

  1. Susceptible Depletion: The susceptible pool shrinks, reducing transmission rates.
  2. Logistic Saturation: Growth transitions from exponential to logistic (S-shaped).
  3. Generation Structure: The age-of-infection distribution becomes non-stationary.

Examples from this analysis:

  • By D=10 days, Omicron's Z reaches 43.82% of N, far beyond the ~5% threshold for valid exponential growth.
  • By D=7 days, Omicron's Z is 14.1% of N, already in the regime where depletion effects matter significantly.

Implications:

To truly satisfy both constraints (≥2 gens with <5% depletion), the population must be much larger than 10,000. Alternatively, a more sophisticated model accounting for susceptible dynamics must be used.

Recommendations

Recommendation 1: If N=10,000 is Fixed

  1. Use D = 5-7 days as a practical compromise. This spans ~2 generations for Influenza and keeps depletion under 15% for most pathogens.
  2. Validate findings against simulations. Run stochastic epidemic simulations (e.g., SEIR model) with N=10,000 to verify that growth rates estimated from 5–7 day windows remain unbiased when susceptible depletion is accounted for.
  3. Document limitations. Clearly state that the analysis uses an exponential model valid only in early epidemic phases, and that results may diverge from reality when depletion is >5%.

Recommendation 2: If Study Design Is Flexible

  1. Increase population size to N ≥ 100,000. This would allow D~10–14 days with <5% depletion across all pathogens, while spanning 1.4–2 generations.
  2. Alternatively, use age-cohort-specific analysis. Instead of whole-population tracking, focus on narrow age groups with high transmission intensity. This reduces the effective N and allows shorter windows.

Recommendation 3: Best Practise for Analysis

  1. Use D = 7 days as the primary analysis window (balance between generation coverage and model validity).
  2. Run sensitivity analyses with D ∈ {5, 7, 10} days to assess robustness of estimates.
  3. Report both growth rates and confidence intervals from each window, acknowledging that wider windows (D=10) may have higher depletion bias but lower sampling variance.
  4. Compare to compartmental models (e.g., logistic, SEIR) that explicitly model susceptible depletion, to bound the magnitude of bias.
