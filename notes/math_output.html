Deterministic epidemic trajectory for a Gamma-distributed generation interval

Renewal equation for arbitrary generation intervals

With a general generation interval distribution \(g(\tau)\), the deterministic epidemic is governed by the renewal equation, not an ODE:

$$j(t) = S(t) \cdot \int_0^t R_0 \, g(\tau) \, j(t - \tau) \, d\tau$$

where \(j(t) = dC/dt\) is the incidence rate, \(S(t) = 1 - C(t)/N\) is the susceptible fraction, and \(g(\tau)\) is the generation interval density — here \(\text{Gamma}(\alpha_{\text{pop}},\, r)\) with \(r = \alpha_{\text{pop}}/T\).

This is a Volterra integral equation of the second kind. There are two natural approaches for Gamma generation intervals:

Option 1: Discretised renewal equation

Discretise time with step \(\Delta t\). Then at each time step \(i\):

$$j_i = S_i \cdot R_0 \sum_{k=1}^{i} g(k \Delta t) \, j_{i-k} \, \Delta t, \qquad C_i = C_{i-1} + j_i \, \Delta t, \qquad S_i = 1 - C_i/N$$

where \(g(k\Delta t) = f_{\text{Gamma}}(k\Delta t;\, \alpha_{\text{pop}},\, r)\). Seed with \(j_0 = 1/N\). This is:

  • General: works for any \(\alpha_{\text{pop}}\) (integer or not), and indeed any \(g\)
  • Simple: a single for loop in R, ~10 lines
  • Accurate: with \(\Delta t = 0.01\) it's essentially exact
  • Requires \(O(n^2)\) work (convolution at each step), but for \(N = 1000\) and \(T_{\max} \sim 100\) this is instant

Option 2: Linear chain trick (ODE, integer \(\alpha_{\text{pop}}\) only)

When \(\alpha_{\text{pop}} = n\) is a positive integer, the Gamma distribution is an Erlang, and we can convert the integral equation to an ODE system. The key observation is that for \(n\) sequential stages each with rate \(r\):

$$P(\text{in stage } n \text{ at time } \tau) = \frac{e^{-r\tau}(r\tau)^{n-1}}{(n-1)!}$$

and the Gamma density satisfies \(g(\tau) = r \cdot P(\text{in stage } n \text{ at } \tau)\). So if an individual transmits only while in the last stage, at rate \(\beta = R_0 \cdot r\), the generation interval is exactly \(\text{Gamma}(n, r)\).

The ODE system has \(n\) sequential stages \(I_1, \ldots, I_n\), with transmission from \(I_n\) only:

$$\frac{dS}{dt} = -\beta \, I_n \, S$$ $$\frac{dI_1}{dt} = \beta \, I_n \, S - r \, I_1$$ $$\frac{dI_k}{dt} = r \, I_{k-1} - r \, I_k, \qquad k = 2, \ldots, n$$ $$\frac{dR}{dt} = r \, I_n$$

with \(\beta = R_0 \cdot r\), initial conditions \(I_1(0) = 1/N\), all others zero. This is equivalent to: stages 1 through \(n-1\) are "latent" (transitioning at rate \(r\) each), and stage \(n\) is "infectious" (transmitting at rate \(\beta\) and recovering at rate \(r\)). The generation interval is \(\text{Gamma}(n-1, r) + \text{Exp}(r) = \text{Gamma}(n, r)\). ✓

For \(\alpha_{\text{pop}} = 10\), this gives a 10-compartment ODE — perfectly tractable with odin.

Recommendation

The discretised renewal equation is the better choice here because:

  1. It works for any \(\alpha_{\text{pop}}\) (you may want non-integer values later)
  2. It's trivial to implement — no ODE solver or odin compilation needed
  3. It's transparent: the generation interval \(g(\tau)\) appears explicitly
  4. It generalises immediately if you change the generation interval distribution

A minimal R implementation:

renewal_epidemic <- function(R0, popshape, T_gen, N, dt = 0.01, tmax = 100) {
  r <- popshape / T_gen
  times <- seq(0, tmax, by = dt)
  nt <- length(times)

  j <- numeric(nt)       # incidence rate
  C <- numeric(nt)       # cumulative infections
  j[1] <- 1 / N          # seed
  C[1] <- 1 / N

  # Pre-compute generation interval weights
  g <- dgamma(times, shape = popshape, rate = r) * dt

  for (i in 2:nt) {
    S <- 1 - C[i-1]
    # Convolution: sum over past incidence weighted by g
    j[i] <- S * R0 * sum(g[2:i] * j[(i-1):1])
    C[i] <- C[i-1] + j[i] * dt
  }

  tibble(t = times, j = j, cuminf = C, S = 1 - C)
}
Initial conditions for the renewal equation: matching "one person infected at t=0"

The delta-function seed

The renewal equation for the incidence rate \(j(t)\) (as a fraction of the population) seeded by a single index case at \(t = 0\) is:

$$j(t) = S(t) \cdot R_0 \left[\frac{g(t)}{N} + \int_0^t g(\tau)\, j(t - \tau)\, d\tau \right]$$

The two terms have different origins:

  • Index case: \(g(t)/N\) — one person (fraction \(1/N\)) infected at \(t = 0\) produces offspring at rate \(R_0 g(\tau)\) at time \(\tau\). No \(d\tau\) here because it's a single discrete person, not a continuous flow.
  • All subsequent cases: \(\int g(\tau)\, j(t-\tau)\, d\tau\) — the \(j(t-\tau)\,d\tau\) fraction infected during \([t-\tau, t-\tau+d\tau]\) produces offspring via the same kernel.

What went wrong

The current code sets j[1] = 1/N and lets the convolution handle everything:

$$j_i = S_i \cdot R_0 \sum_{k} \underbrace{g(k\Delta t) \cdot \Delta t}_{= \texttt{g[k]}} \cdot\; j_{i-k}$$

When the index case enters this sum at lag \(k = i\), its contribution is:

$$g(i\Delta t) \cdot \Delta t \cdot \frac{1}{N} = \frac{g(i\Delta t)}{N} \cdot \Delta t$$

But the correct contribution is \(g(i\Delta t)/N\) — without the extra \(\Delta t\). The index case is a delta function \(\delta(t)/N\) in incidence, not a continuous rate \(1/N\). By treating it as a rate, the seed is effectively \(\Delta t = 0.01\) times too weak. Each subsequent generation inherits this factor, so the entire trajectory is delayed by roughly \(\log(1/\Delta t)/\alpha \approx 4.6\) days.

The fix

Separate the index case from the convolution explicitly:

renewal_epidemic <- function(R0, popshape, T_gen, N, dt = 0.01, tmax = 100) {
  r <- popshape / T_gen
  times <- seq(0, tmax, by = dt)
  nt <- length(times)

  j <- numeric(nt)
  C <- numeric(nt)
  C[1] <- 1 / N

  # Generation interval weights (with dt baked in for the convolution)
  g <- dgamma(times, shape = popshape, rate = r) * dt
  # Index case kernel (no dt — it's a single person, not a rate)
  g_index <- dgamma(times, shape = popshape, rate = r) / N

  for (i in 2:nt) {
    S <- 1 - C[i-1]
    # Index case + renewal convolution
    j[i] <- S * R0 * (g_index[i] + sum(g[2:i] * j[(i-1):1]))
    C[i] <- C[i-1] + j[i] * dt
  }

  tibble(t = times, j = j, cuminf = C, S = 1 - C)
}

This is equivalent to setting j[1] = 1/(N*dt) (the discretised delta function), but makes the reasoning explicit.

Why does κ=0.25 appear to be the slowest to reach 100 cases?

The survival curves cross, but κ=0 is still slowest overall

For the Gamma model (\(\alpha_{\text{pop}} = 10,\ T = 5,\ R_0 = 12\)), the \(W\) distribution at each \(\kappa\) is approximately Gamma with shape \(k = 1/\text{Var}(W)\), rate \(k\), mean 1:

The ordering is monotonic: \(\kappa = 0\) (spike) has the latest median and mean milestone time, as expected. But the difference between \(\kappa = 0\) and \(\kappa = 0.25\) is tiny — only ~0.15 days in the median.

Why it looks reversed with 100 simulations

The milestone time has substantial stochastic spread. For a Gamma\((k, k)\) distribution of \(W\), the variance of \(\log W\) is \(\psi_1(k)\) (the trigamma function), so:

$$\text{SD}(T_{100}) \;\approx\; \frac{\sqrt{\psi_1(k)}}{\alpha}$$

This gives SD \(\approx 2.2\) days for \(\kappa = 0\) and \(\approx 1.8\) days for \(\kappa = 0.25\). With \(n_{\text{sim}} = 100\), the standard error of the median is:

$$\text{SE}_{\text{median}} \approx \frac{\sqrt{\pi/2}}{\sqrt{n}} \cdot \text{SD} \;\approx\; 0.25 \text{ days}$$

So the SE of the difference in medians is \(\approx 0.35\) days. The true difference (0.15 days) is only 0.4 standard errors — well within noise. The apparent reversal between \(\kappa = 0\) and \(\kappa = 0.25\) is a simulation artefact that will disappear with more runs.

The crossing mechanism is real, though

Your intuition about the variance effect is correct — the \(\kappa=0\) and \(\kappa=0.25\) survival curves genuinely cross (at \(t \approx 3.5\) and \(t \approx 7.8\)). In the window between these crossings, \(\kappa=0\) has lower survival (appears faster) because its heavy right tail of \(W\) produces a handful of very fast epidemics. But this crossing isn't strong enough to make \(\kappa=0.25\) the slowest overall — it just means the two curves interleave.

The underlying reason the differences are so small: \(\alpha_{\text{pop}} = 10\) makes the generation interval quite tightly concentrated (CV = 0.32), which limits how much the punctuation parameter \(\kappa\) can spread out \(W\). With a less concentrated generation interval (smaller \(\alpha_{\text{pop}}\)), the separation between \(\kappa\) values would be much more dramatic.

Gamma approximation to the SEIR generation interval (e_dur=2, i_dur=3)

Moment-matching the SEIR generation interval

In the SEIR ODE, a person infected at time 0 spends \(\text{Exp}(\nu)\) time latent, then transmits at constant rate \(\beta\) for an \(\text{Exp}(\gamma)\) infectious period. The population-level infectiousness profile is:

$$A(\tau) = \beta \cdot P(\text{in } I \text{ at time } \tau) = \beta \cdot \frac{\nu}{\nu - \gamma}\left(e^{-\gamma\tau} - e^{-\nu\tau}\right)$$

with \(\nu = 1/2,\ \gamma = 1/3\). Since \(\int_0^\infty A(\tau)\,d\tau = R_0\), the generation interval density is:

$$g(\tau) = \frac{A(\tau)}{R_0} = e^{-\tau/3} - e^{-\tau/2}$$

This is a hypoexponential density — the distribution of \(T_E + T_I\) where \(T_E \sim \text{Exp}(1/2)\) and \(T_I \sim \text{Exp}(1/3)\) are independent. Its moments:

$$E[\tau] = 2 + 3 = 5, \qquad \text{Var}(\tau) = 4 + 9 = 13, \qquad \text{CV} = \frac{\sqrt{13}}{5} \approx 0.72$$

Gamma moment-match

Matching \(\text{Gamma}(\alpha_{\text{pop}},\, r)\) to these two moments: mean \(= \alpha_{\text{pop}}/r = 5\), variance \(= \alpha_{\text{pop}}/r^2 = 13\). Solving:

$$\alpha_{\text{pop}} = \frac{(\text{mean})^2}{\text{var}} = \frac{25}{13} \approx 1.92, \qquad r = \frac{\text{mean}}{\text{var}} = \frac{5}{13} \approx 0.385$$

Why does κ=0.25 look so similar to the spike (κ=0)? [Measles parameters]

Var(W) is highly convex in κ

Your thinking is right that \(\kappa = 0.25\) makes the individual infectiousness profile's SD equal to half the population GI's SD (\(\sqrt{\kappa} = 0.5\)). But the effect on Var(\(W\)) is highly nonlinear — the extra variance beyond the smooth baseline drops like \(R_0^{-2\kappa}\), which is convex and front-loaded.

The key formula

The extra variance beyond the smooth limit decomposes as:

$$\Delta(\kappa) \;=\; \text{Var}(W)\big|_\kappa - \text{Var}(W)\big|_{\kappa=1} \;=\; \frac{R_0^2}{1 - m_2}\;\underbrace{\rho^{2\kappa\alpha_{\text{pop}}}}_{\text{prefactor}} \cdot \underbrace{\left(\rho_2^{(1-\kappa)\alpha_{\text{pop}}} - \rho^{2(1-\kappa)\alpha_{\text{pop}}}\right)}_{\text{bracket}}$$

Since \(\rho^{\alpha_{\text{pop}}} = 1/R_0\), the prefactor simplifies to:

$$\rho^{2\kappa\alpha_{\text{pop}}} = R_0^{-2\kappa}$$

This factor is exponential in \(\kappa\) and drops very steeply near \(\kappa = 0\):

\(\kappa\)\(R_0^{-2\kappa}\)Var(\(W\))% of spike extra var
01.0000.798100%
0.100.6080.72088%
0.250.2890.60871%
0.500.0830.43844%
0.750.0240.28621%
10.0070.1500%

At \(\kappa = 0.25\), the individual profile has half the SD of the population GI — but 71% of the spike's extra variance in \(W\) is retained. You'd need roughly \(\kappa \approx 0.5\) for a halfway effect.

Intuition

The \(R_0^{-2\kappa}\) prefactor comes from the Laplace transform of the jitter component evaluated at \(2\alpha\). The jitter's shape is \(\kappa \alpha_{\text{pop}}\), and even a small shape parameter produces substantial "diversification" — reducing variance by spreading offspring times apart. But the marginal diversification benefit diminishes rapidly: the first 25% of jitter variance eliminates only 29% of the extra W-variance, while the last 25% eliminates 21%.

The effect is stronger at higher \(R_0\) because \(R_0^{-2\kappa}\) drops faster. For \(R_0 = 2\) (influenza), \(R_0^{-0.5} = 0.71\) — a modest drop. For \(R_0 = 12\) (measles), \(R_0^{-0.5} = 0.29\) — a steep drop but still leaving most of the extra variance. The net effect is that \(\kappa = 0.25\) stays close to the spike across all realistic \(R_0\).

Derivation of Var(W) = (cα + σ(1−ψ)α − 1) / (1 − cα)

Deriving the variance of the CMJ limit W

Setup

Each infector produces \(N \sim \text{Poisson}(R_0)\) offspring. Offspring \(j\) is infected at time \(\tau_j = s + \varepsilon_j\), where:

  • \(s \sim \text{Gamma}((1-\psi)\alpha,\, r)\) — shared shift, common to all offspring of this infector
  • \(\varepsilon_j \sim \text{Gamma}(\psi\alpha,\, r)\) — independent jitter, iid across offspring

Marginally, \(\tau_j \sim \text{Gamma}(\alpha, r)\) (by the additivity of Gamma distributions with the same rate).

The Malthusian parameter \(\alpha_M\) satisfies the Euler–Lotka equation:

$$R_0 \cdot E[e^{-\alpha_M \tau}] = 1$$

Define \(\rho = \frac{r}{r + \alpha_M}\), so \(E[e^{-\alpha_M \tau}] = \rho^\alpha\) and the Euler–Lotka equation becomes \(R_0 \rho^\alpha = 1\), i.e. \(\rho^\alpha = 1/R_0\).

Step 1: The branching recursion for W

The CMJ limit \(W = \lim_{t \to \infty} e^{-\alpha_M t} I_b(t)\) satisfies the distributional fixed-point equation:

$$W \stackrel{d}{=} \sum_{j=1}^{N} e^{-\alpha_M \tau_j} W_j$$

where the \(W_j\) are iid copies of \(W\), independent of \((N, s, \varepsilon_1, \ldots)\). Define the discounted offspring sum:

$$S = \sum_{j=1}^{N} e^{-\alpha_M \tau_j}$$

By the Euler–Lotka equation, \(E[S] = R_0 \cdot E[e^{-\alpha_M \tau}] = 1\), and from the fixed point, \(E[W] = E[S] \cdot E[W] = 1\) (starting from \(I_0 = 1\)).

Step 2: Var(W) in terms of Var(S)

From the fixed-point equation, compute \(E[W^2]\) by conditioning on \((N, s, \varepsilon_1, \ldots, \varepsilon_N)\):

$$E[W^2 \mid N, s, \boldsymbol{\varepsilon}] = \sum_j e^{-2\alpha_M \tau_j} E[W^2] + \sum_{j \ne k} e^{-\alpha_M(\tau_j + \tau_k)}$$

Recognising that \(S^2 = \sum_j e^{-2\alpha_M \tau_j} + \sum_{j \ne k} e^{-\alpha_M(\tau_j + \tau_k)}\), this becomes:

$$E[W^2 \mid N, s, \boldsymbol{\varepsilon}] = (E[W^2] - 1) \sum_j e^{-2\alpha_M \tau_j} + S^2$$

Taking expectations and defining \(m_2 = E\!\left[\sum_j e^{-2\alpha_M \tau_j}\right] = R_0 \rho_2^\alpha\) where \(\rho_2 = \frac{r}{r + 2\alpha_M}\):

$$E[W^2] = (E[W^2] - 1)\, m_2 + E[S^2]$$

Rearranging, and using \(E[W] = E[S] = 1\):

$$\boxed{\text{Var}(W) = \frac{\text{Var}(S)}{1 - m_2}}$$

This is valid whenever \(m_2 < 1\), which holds in the supercritical regime.

Step 3: Computing E[S²] by conditioning on (N, s)

Factor the offspring times as \(\tau_j = s + \varepsilon_j\):

$$S = e^{-\alpha_M s} \sum_{j=1}^{N} e^{-\alpha_M \varepsilon_j}$$ $$S^2 = e^{-2\alpha_M s} \left(\sum_{j=1}^{N} e^{-\alpha_M \varepsilon_j}\right)^2$$

Condition on \(N\) and \(s\). The \(\varepsilon_j\) are iid \(\text{Gamma}(\psi\alpha, r)\), so:

\begin{align} E[e^{-\alpha_M \varepsilon}] &= \rho^{\psi\alpha} \\ E[e^{-2\alpha_M \varepsilon}] &= \rho_2^{\psi\alpha} \end{align}

Expanding the square:

$$E\!\left[\left(\sum_j e^{-\alpha_M \varepsilon_j}\right)^2 \,\middle|\, N\right] = N\, \rho_2^{\psi\alpha} + N(N-1)\, \rho^{2\psi\alpha}$$

Taking expectation over \(N \sim \text{Poisson}(R_0)\) (using \(E[N] = R_0\), \(E[N(N-1)] = R_0^2\)):

$$E\!\left[\left(\sum_j e^{-\alpha_M \varepsilon_j}\right)^2\right] = R_0\, \rho_2^{\psi\alpha} + R_0^2\, \rho^{2\psi\alpha}$$

Now take expectation over \(s \sim \text{Gamma}((1-\psi)\alpha, r)\), using \(E[e^{-2\alpha_M s}] = \rho_2^{(1-\psi)\alpha}\):

$$E[S^2] = \rho_2^{(1-\psi)\alpha}\!\left[R_0\, \rho_2^{\psi\alpha} + R_0^2\, \rho^{2\psi\alpha}\right] = R_0\, \rho_2^\alpha + R_0^2\, \rho^{2\psi\alpha}\, \rho_2^{(1-\psi)\alpha}$$

Since \(E[S] = 1\):

$$\text{Var}(S) = R_0\, \rho_2^\alpha + R_0^2\, \rho^{2\psi\alpha}\, \rho_2^{(1-\psi)\alpha} - 1$$

Step 4: Introducing c and σ

Define \(\mu = R_0^{1/\alpha}\), the per-stage growth factor. From \(\rho^\alpha = 1/R_0\), we get \(\rho = 1/\mu\), so \(\alpha_M = (\mu - 1)r\). Then:

$$\rho_2 = \frac{r}{r + 2\alpha_M} = \frac{r}{(2\mu - 1)r} = \frac{1}{2\mu - 1}$$

Define two key ratios:

$$c = \frac{\rho_2}{\rho} = \frac{\mu}{2\mu - 1}, \qquad \sigma = \frac{\rho_2}{\rho^2} = \frac{\mu^2}{2\mu - 1} = \mu \cdot c$$

Both depend only on \(\mu = R_0^{1/\alpha}\), not on \(\psi\).

Express the terms:

First term: \(m_2 = R_0\, \rho_2^\alpha = \rho_2^\alpha / \rho^\alpha = (\rho_2/\rho)^\alpha = c^\alpha\).

Second term: Using \(\rho^{2\psi\alpha} = (\rho^\alpha)^{2\psi} = R_0^{-2\psi}\):

$$R_0^2\, \rho^{2\psi\alpha}\, \rho_2^{(1-\psi)\alpha} = R_0^{2(1-\psi)}\, \rho_2^{(1-\psi)\alpha} = \left(\frac{\rho_2}{\rho^2}\right)^{(1-\psi)\alpha} = \sigma^{(1-\psi)\alpha}$$

where we used \(\rho^{2(1-\psi)\alpha} = R_0^{-2(1-\psi)}\) to write \(R_0^{2(1-\psi)} \rho_2^{(1-\psi)\alpha} = (\rho_2/\rho^2)^{(1-\psi)\alpha}\).

Step 5: Final result

Substituting into \(\text{Var}(W) = \text{Var}(S)/(1 - m_2)\):

$$\boxed{\text{Var}(W) = \frac{c^\alpha + \sigma^{(1-\psi)\alpha} - 1}{1 - c^\alpha}}$$

where \(c = \frac{\mu}{2\mu - 1}\), \(\sigma = \frac{\mu^2}{2\mu - 1}\), \(\mu = R_0^{1/\alpha}\).

Key observations

  • Smooth limit (\(\psi = 1\)): \(\sigma^0 = 1\), so \(\text{Var}(W) = c^\alpha/(1 - c^\alpha) = m_2/(1 - m_2)\). This is the minimum variance — each offspring time is fully independent.
  • Spike limit (\(\psi = 0\)): \(\sigma^\alpha = R_0^2 \rho_2^\alpha\), giving \(\text{Var}(W) = (c^\alpha + \sigma^\alpha - 1)/(1 - c^\alpha)\). This is the maximum — all offspring share the same timing.
  • \(\psi\) only enters through \(\sigma^{(1-\psi)\alpha}\), confirming that narrower profiles (smaller \(\psi\)) always increase Var(\(W\)).
  • Since \(E[W] = 1\), we have \(\text{CV}^2(W) = \text{Var}(W)\).
Conditional moments, Gamma approximation, and time-shift mode

From Var(W) to the mode of the time shift

Conditional moments of W

The random variable \(W\) has a point mass at 0 (extinction, probability \(q\)) and a continuous part on \((0, \infty)\) (survival, probability \(1 - q\)). Since all unconditional moments of \(W\) come entirely from the continuous part:

$$E[W^n] = (1 - q)\, E[W^n \mid W > 0]$$

so the conditional (on survival) raw moments are simply:

$$E[W^n \mid \text{surv}] = \frac{E[W^n]}{1 - q}$$

For the extinction probability, \(q\) is the smallest root of \(q = e^{R_0(q - 1)}\) (Poisson offspring). For \(R_0 \gg 1\), \(q \approx 0\).

First two unconditional moments

We already derived \(E[W] = 1\) and:

$$E[W^2] = 1 + \text{Var}(W) = 1 + \frac{c^\alpha + \sigma^{(1-\psi)\alpha} - 1}{1 - c^\alpha} = \frac{\sigma^{(1-\psi)\alpha}}{1 - c^\alpha}$$

where we combined terms: \(1 + (c^\alpha + \sigma^{(1-\psi)\alpha} - 1)/(1 - c^\alpha) = (1 - c^\alpha + c^\alpha + \sigma^{(1-\psi)\alpha} - 1)/(1 - c^\alpha)\).

Conditional CV²

The conditional coefficient of variation squared is:

\begin{align} \text{CV}^2_{\text{cond}} &= \frac{\text{Var}(W \mid \text{surv})}{(E[W \mid \text{surv}])^2} \\[6pt] &= \frac{E[W^2 \mid \text{surv}]}{(E[W \mid \text{surv}])^2} - 1 \\[6pt] &= \frac{E[W^2]/(1-q)}{1/(1-q)^2} - 1 \\[6pt] &= (1 - q)\, E[W^2] - 1 \end{align}

Substituting \(E[W^2] = \sigma^{(1-\psi)\alpha}/(1 - c^\alpha)\):

$$\text{CV}^2_{\text{cond}} = \frac{(1 - q)\, \sigma^{(1-\psi)\alpha}}{1 - c^\alpha} - 1$$

When \(q \approx 0\): \(\text{CV}^2_{\text{cond}} \approx \text{Var}(W)\), since \(E[W] = 1\) makes \(\text{CV}^2 = \text{Var}\) unconditionally.

Two-moment Gamma approximation

Approximate \(W \mid \text{surv}\) by \(\text{Gamma}(k, \lambda)\), matching the first two moments. Write \(V = \text{CV}^2_{\text{cond}}\) and \(p = 1 - q\) for brevity.

Quantity In terms of \(k, \lambda\) Solved
Mean \(k/\lambda = 1/p\) \(k = 1/V\)
CV² \(1/k = V\) \(\lambda = p/V\)

First three moments of the Gamma approximation

For \(\text{Gamma}(k, \lambda)\), the \(n\)-th raw moment is \(\prod_{j=0}^{n-1}(k+j)/\lambda^n\). Using \(k = 1/V\) and \(\lambda = p/V\):

\begin{align} E[W \mid \text{surv}] &= \frac{1}{p} \\[8pt] E[W^2 \mid \text{surv}] &= \frac{1 + V}{p^2} \\[8pt] E[W^3 \mid \text{surv}] &= \frac{(1 + V)(1 + 2V)}{p^3} \end{align}

The first two are exact (matched by construction). The third is a prediction of the Gamma approximation — it will differ from the true \(E[W^3 \mid \text{surv}]\) unless \(W \mid \text{surv}\) happens to be exactly Gamma.

Expanding \(V\) gives the fully explicit forms. For instance:

$$E[W^2 \mid \text{surv}] = \frac{1}{p^2}\left(1 + \frac{p\, \sigma^{(1-\psi)\alpha}}{1 - c^\alpha} - 1\right) + \frac{1}{p^2} = \frac{\sigma^{(1-\psi)\alpha}}{p(1 - c^\alpha)}$$

which matches the exact value \(E[W^2]/(1-q) = \sigma^{(1-\psi)\alpha}/[p(1 - c^\alpha)]\). ✓

Mode of the time shift

The time shift is \(\varsigma = \log(W)/\alpha_M\) (written as \(\log(W)/r\) in the writeup, where \(r\) denotes the growth rate \(\alpha_M\)). Its mode is determined by the mode of \(W \mid \text{surv}\).

The mode of a \(\text{Gamma}(k, \lambda)\) is \((k-1)/\lambda\) (for \(k \geq 1\)). Substituting:

$$\text{mode}(W \mid \text{surv}) = \frac{k - 1}{\lambda} = \frac{1/V - 1}{p/V} = \frac{1 - V}{p}$$

Taking the log and dividing by \(\alpha_M\):

$$\text{mode}_\varsigma = \frac{1}{\alpha_M}\left[\log(1 - V) - \log p\right] = \frac{1}{\alpha_M}\left[\log(1 - \text{CV}^2_{\text{cond}}) - \log(1 - q)\right]$$

When extinction is negligible (\(q \approx 0\), so \(p \approx 1\)):

$$\boxed{\text{mode}_\varsigma \approx \frac{1}{\alpha_M}\log\!\left(1 - \text{CV}^2_{\text{cond}}\right)}$$

which is exactly the expression in the writeup (with the identification \(r \equiv \alpha_M\)).

Interpretation

  • Sign: Since \(V = \text{CV}^2_{\text{cond}} > 0\), the mode is \(\log(1 - V) < 0\), hence \(\text{mode}_\varsigma < 0\). The most likely epidemic is delayed relative to the deterministic trajectory — the mode of \(W\) is less than its mean.
  • Magnitude: Larger \(V\) (more punctuated profile, smaller \(\psi\)) pushes the mode further negative — the typical epidemic starts later. In the extreme, \(V \to 1\) makes \(\text{mode}_\varsigma \to -\infty\).
  • ψ-dependence: Substituting \(V \approx \text{Var}(W) = (c^\alpha + \sigma^{(1-\psi)\alpha} - 1)/(1 - c^\alpha)\), the mode depends on \(\psi\) solely through \(\sigma^{(1-\psi)\alpha}\). The \(\psi = 1\) (smooth) limit gives the least negative mode; \(\psi = 0\) (spike) gives the most negative.
  • Skewness: The Gamma approximation has skewness \(2/\sqrt{k} = 2\sqrt{V}\). More punctuated profiles increase \(V\), making \(W\) more right-skewed: a few epidemics start very early, but the bulk are delayed.

Validity constraint

The mode formula requires \(k \geq 1\), i.e. \(V \leq 1\), i.e. \(\text{CV}^2_{\text{cond}} \leq 1\). When \(V > 1\), the Gamma density is monotonically decreasing (mode at 0), and the mode of the time shift is \(-\infty\) — the most likely outcome becomes extinction or near-extinction. This occurs for very punctuated profiles with moderate \(R_0\). For the spike limit (\(\psi = 0\)) with \(q \approx 0\):

$$V_{\text{spike}} = \frac{c^\alpha + \sigma^\alpha - 1}{1 - c^\alpha} = \frac{c^\alpha + R_0^2 \rho_2^\alpha - 1}{1 - c^\alpha}$$

This exceeds 1 when \(\sigma^\alpha > 2(1 - c^\alpha)\), which can happen for small \(R_0\) or small \(\alpha\).

Overdispersion from periodic contacts × punctuated infectiousness

Deriving the dispersion parameter k

Setup

Consider a population with periodic contact rate:

$$c(t) = \bar{c} + A\sin(\omega t)$$

where \(\bar{c}\) is the mean contact rate, \(A < \bar{c}\) is the amplitude, and \(\omega = 2\pi/P\) is the angular frequency (period \(P\)). An individual infected at calendar time \(t_0\) has a biological infectiousness profile \(a_i(\tau)\) — a density over time-since-infection \(\tau\). Their individual reproduction number is:

$$\mathcal{R}_i = R_0 \int_0^\infty a_i(\tau)\,\frac{c(t_0 + \tau)}{\bar{c}}\, d\tau$$

In the punctuated model, \(a_i(\tau)\) is the density of \(s_i + \varepsilon\) where \(\varepsilon \sim \text{Gamma}(\psi\alpha, r)\), centred on the individual's shared shift \(s_i\). The jitter \(\varepsilon\) spreads the profile, while \(s_i\) locates it in time.

Step 1: Characteristic function argument

Substitute \(c(t_0 + \tau)/\bar{c} = 1 + (A/\bar{c})\sin(\omega(t_0 + \tau))\):

$$\mathcal{R}_i = R_0\!\left[1 + \frac{A}{\bar{c}}\int_0^\infty a_i(\tau)\sin(\omega(t_0 + \tau))\,d\tau\right]$$

The integral is the imaginary part of the characteristic function of \(a_i\) at frequency \(\omega\). Since \(a_i\) is the density of \(s_i + \varepsilon\):

$$\int a_i(\tau)\, e^{i\omega\tau}\, d\tau = e^{i\omega s_i} \cdot \underbrace{\left(\frac{r}{r - i\omega}\right)^{\psi\alpha}}_{\text{char. fn. of } \varepsilon}$$

Write the Gamma characteristic function in polar form:

$$\left(\frac{r}{r - i\omega}\right)^{\psi\alpha} = D\, e^{i\varphi}$$

where the damping factor is:

$$D = \left|\frac{r}{r - i\omega}\right|^{\psi\alpha} = \left(1 + \frac{\omega^2}{r^2}\right)^{-\psi\alpha/2}$$

and \(\varphi = \psi\alpha\cdot\arctan(\omega/r)\) is a phase shift. Then:

$$\int a_i(\tau)\sin(\omega(t_0 + \tau))\,d\tau = D\cdot\sin(\omega(t_0 + s_i) + \varphi)$$

So each individual's reproduction number is:

$$\mathcal{R}_i = R_0\!\left[1 + \frac{A}{\bar{c}}\, D\,\sin\!\big(\omega(t_0 + s_i) + \varphi\big)\right]$$

Step 2: Variance of \(\mathcal{R}_i\)

The phase \(\omega(t_0 + s_i) + \varphi\) varies across individuals (through both infection time \(t_0\) and shared shift \(s_i\)). Averaging over the population — either because \(t_0\) is uniformly distributed over the cycle, or because \(s_i\) has enough spread — \(\sin^2\) averages to 1/2:

$$E[\mathcal{R}_i] = R_0, \qquad \text{Var}(\mathcal{R}_i) = R_0^2 \cdot \frac{A^2}{\bar{c}^2} \cdot D^2 \cdot \frac{1}{2}$$

Step 3: The dispersion parameter

Each individual produces \(N_i \sim \text{Poisson}(\mathcal{R}_i)\) offspring, so:

$$\text{Var}(N_i) = E[\mathcal{R}_i] + \text{Var}(\mathcal{R}_i) = R_0 + \frac{R_0^2 A^2 D^2}{2\bar{c}^2}$$

Matching to the negative binomial mean-variance relation \(\text{Var} = \mu + \mu^2/k\):

$$k = \frac{R_0^2}{\text{Var}(\mathcal{R}_i)} = \frac{2\bar{c}^2}{A^2 D^2}$$

Substituting \(D^2 = (1 + \omega^2/r^2)^{-\psi\alpha}\):

$$\boxed{k = \frac{2\bar{c}^2}{A^2}\left(1 + \frac{\omega^2}{r^2}\right)^{\psi\alpha}}$$

Here \(r = \alpha/T\) is the Gamma rate parameter of the generation interval (not the epidemic growth rate).

Note: The expression in uncontrolled.tex line 37 has a LaTeX typo: \frac{2}{\bar{c}^2}{A^2} renders as \(2A^2/\bar{c}^2\), but the correct coefficient is \(2\bar{c}^2/A^2\). (Sanity check: \(k \to \infty\) as \(A \to 0\), recovering Poisson.)

Step 4: The critical period

Define the critical period:

$$z^*_{\text{per}} = \frac{2\pi}{r} = \frac{2\pi T}{\alpha}$$

This is the period of a contact cycle whose angular frequency matches the Gamma rate \(r\). Since \(\omega = 2\pi/z_{\text{per}}\) where \(z_{\text{per}} = P\) is the contact period:

$$\frac{\omega}{r} = \frac{z^*_{\text{per}}}{z_{\text{per}}}$$

and the formula becomes:

$$k = \frac{2\bar{c}^2}{A^2}\left(1 + \left(\frac{z^*_{\text{per}}}{z_{\text{per}}}\right)^{\!2}\;\right)^{\psi\alpha}$$

Three regimes

Regime Condition \(k\) Interpretation
Slow cycle \(z_{\text{per}} \gg z^*_{\text{per}}\) \(\approx 2\bar{c}^2/A^2\) Contact cycle so slow that even jittered offspring see a nearly constant \(c\). Overdispersion maximal, independent of \(\psi\).
Critical \(z_{\text{per}} \approx z^*_{\text{per}}\) \(\approx (2\bar{c}^2/A^2)\cdot 2^{\psi\alpha}\) Maximum sensitivity to \(\psi\): smooth profiles average out fluctuations, punctuated profiles don't.
Fast cycle \(z_{\text{per}} \ll z^*_{\text{per}}\) \(\sim (z^*_{\text{per}}/z_{\text{per}})^{2\psi\alpha} \to \infty\) Offspring average over many cycles → Poisson. Unless \(\psi = 0\) (spike), where \(k = 2\bar{c}^2/A^2\) always.

Physical intuition

The key insight is that the shared shift \(s\) does not reduce overdispersion — it merely relocates the burst to a different phase of the contact cycle, but all offspring still see the same contact rate. Only the jitter \(\varepsilon\) (controlled by \(\psi\)) averages over the contact fluctuations, because it spreads offspring across different phases.

  • Spike (\(\psi = 0\)): No jitter, \(D = 1\). All offspring hit the contact cycle at the same phase. \(k = 2\bar{c}^2/A^2\), maximal overdispersion regardless of period.
  • Smooth (\(\psi = 1\)): Full jitter with shape \(\alpha\). \(D = (1 + \omega^2/r^2)^{-\alpha/2}\), strong damping of contact fluctuations. \(k\) grows as \((z^*_{\text{per}}/z_{\text{per}})^{2\alpha}\) for fast cycles.
  • Intermediate \(\psi\): Jitter shape is \(\psi\alpha\), giving partial damping. The exponent \(\psi\alpha\) interpolates between the two limits.

Numerical examples

For measles (\(\alpha = 11.36\), \(T = 12.2\) days): \(z^*_{\text{per}} = 2\pi \cdot 12.2/11.36 \approx 6.7\) days.

For influenza (\(\alpha = 2.32\), \(T = 3.2\) days): \(z^*_{\text{per}} = 2\pi \cdot 3.2/2.32 \approx 8.7\) days.

Both are near one week — so weekly contact patterns sit squarely in the critical regime where the punctuation parameter \(\psi\) has maximal effect on overdispersion.
