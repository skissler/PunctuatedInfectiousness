Deterministic epidemic trajectory for a Gamma-distributed generation interval

Renewal equation for arbitrary generation intervals

With a general generation interval distribution \(g(\tau)\), the deterministic epidemic is governed by the renewal equation, not an ODE:

$$j(t) = S(t) \cdot \int_0^t R_0 \, g(\tau) \, j(t - \tau) \, d\tau$$

where \(j(t) = dC/dt\) is the incidence rate, \(S(t) = 1 - C(t)/N\) is the susceptible fraction, and \(g(\tau)\) is the generation interval density — here \(\text{Gamma}(\alpha_{\text{pop}},\, r)\) with \(r = \alpha_{\text{pop}}/T\).

This is a Volterra integral equation of the second kind. There are two natural approaches for Gamma generation intervals:

Option 1: Discretised renewal equation

Discretise time with step \(\Delta t\). Then at each time step \(i\):

$$j_i = S_i \cdot R_0 \sum_{k=1}^{i} g(k \Delta t) \, j_{i-k} \, \Delta t, \qquad C_i = C_{i-1} + j_i \, \Delta t, \qquad S_i = 1 - C_i/N$$

where \(g(k\Delta t) = f_{\text{Gamma}}(k\Delta t;\, \alpha_{\text{pop}},\, r)\). Seed with \(j_0 = 1/N\). This is:

  • General: works for any \(\alpha_{\text{pop}}\) (integer or not), and indeed any \(g\)
  • Simple: a single for loop in R, ~10 lines
  • Accurate: with \(\Delta t = 0.01\) it's essentially exact
  • Requires \(O(n^2)\) work (convolution at each step), but for \(N = 1000\) and \(T_{\max} \sim 100\) this is instant

Option 2: Linear chain trick (ODE, integer \(\alpha_{\text{pop}}\) only)

When \(\alpha_{\text{pop}} = n\) is a positive integer, the Gamma distribution is an Erlang, and we can convert the integral equation to an ODE system. The key observation is that for \(n\) sequential stages each with rate \(r\):

$$P(\text{in stage } n \text{ at time } \tau) = \frac{e^{-r\tau}(r\tau)^{n-1}}{(n-1)!}$$

and the Gamma density satisfies \(g(\tau) = r \cdot P(\text{in stage } n \text{ at } \tau)\). So if an individual transmits only while in the last stage, at rate \(\beta = R_0 \cdot r\), the generation interval is exactly \(\text{Gamma}(n, r)\).

The ODE system has \(n\) sequential stages \(I_1, \ldots, I_n\), with transmission from \(I_n\) only:

$$\frac{dS}{dt} = -\beta \, I_n \, S$$ $$\frac{dI_1}{dt} = \beta \, I_n \, S - r \, I_1$$ $$\frac{dI_k}{dt} = r \, I_{k-1} - r \, I_k, \qquad k = 2, \ldots, n$$ $$\frac{dR}{dt} = r \, I_n$$

with \(\beta = R_0 \cdot r\), initial conditions \(I_1(0) = 1/N\), all others zero. This is equivalent to: stages 1 through \(n-1\) are "latent" (transitioning at rate \(r\) each), and stage \(n\) is "infectious" (transmitting at rate \(\beta\) and recovering at rate \(r\)). The generation interval is \(\text{Gamma}(n-1, r) + \text{Exp}(r) = \text{Gamma}(n, r)\). ✓

For \(\alpha_{\text{pop}} = 10\), this gives a 10-compartment ODE — perfectly tractable with odin.

Recommendation

The discretised renewal equation is the better choice here because:

  1. It works for any \(\alpha_{\text{pop}}\) (you may want non-integer values later)
  2. It's trivial to implement — no ODE solver or odin compilation needed
  3. It's transparent: the generation interval \(g(\tau)\) appears explicitly
  4. It generalises immediately if you change the generation interval distribution

A minimal R implementation:

renewal_epidemic <- function(R0, popshape, T_gen, N, dt = 0.01, tmax = 100) {
  r <- popshape / T_gen
  times <- seq(0, tmax, by = dt)
  nt <- length(times)

  j <- numeric(nt)       # incidence rate
  C <- numeric(nt)       # cumulative infections
  j[1] <- 1 / N          # seed
  C[1] <- 1 / N

  # Pre-compute generation interval weights
  g <- dgamma(times, shape = popshape, rate = r) * dt

  for (i in 2:nt) {
    S <- 1 - C[i-1]
    # Convolution: sum over past incidence weighted by g
    j[i] <- S * R0 * sum(g[2:i] * j[(i-1):1])
    C[i] <- C[i-1] + j[i] * dt
  }

  tibble(t = times, j = j, cuminf = C, S = 1 - C)
}
Initial conditions for the renewal equation: matching "one person infected at t=0"

The delta-function seed

The renewal equation for the incidence rate \(j(t)\) (as a fraction of the population) seeded by a single index case at \(t = 0\) is:

$$j(t) = S(t) \cdot R_0 \left[\frac{g(t)}{N} + \int_0^t g(\tau)\, j(t - \tau)\, d\tau \right]$$

The two terms have different origins:

  • Index case: \(g(t)/N\) — one person (fraction \(1/N\)) infected at \(t = 0\) produces offspring at rate \(R_0 g(\tau)\) at time \(\tau\). No \(d\tau\) here because it's a single discrete person, not a continuous flow.
  • All subsequent cases: \(\int g(\tau)\, j(t-\tau)\, d\tau\) — the \(j(t-\tau)\,d\tau\) fraction infected during \([t-\tau, t-\tau+d\tau]\) produces offspring via the same kernel.

What went wrong

The current code sets j[1] = 1/N and lets the convolution handle everything:

$$j_i = S_i \cdot R_0 \sum_{k} \underbrace{g(k\Delta t) \cdot \Delta t}_{= \texttt{g[k]}} \cdot\; j_{i-k}$$

When the index case enters this sum at lag \(k = i\), its contribution is:

$$g(i\Delta t) \cdot \Delta t \cdot \frac{1}{N} = \frac{g(i\Delta t)}{N} \cdot \Delta t$$

But the correct contribution is \(g(i\Delta t)/N\) — without the extra \(\Delta t\). The index case is a delta function \(\delta(t)/N\) in incidence, not a continuous rate \(1/N\). By treating it as a rate, the seed is effectively \(\Delta t = 0.01\) times too weak. Each subsequent generation inherits this factor, so the entire trajectory is delayed by roughly \(\log(1/\Delta t)/\alpha \approx 4.6\) days.

The fix

Separate the index case from the convolution explicitly:

renewal_epidemic <- function(R0, popshape, T_gen, N, dt = 0.01, tmax = 100) {
  r <- popshape / T_gen
  times <- seq(0, tmax, by = dt)
  nt <- length(times)

  j <- numeric(nt)
  C <- numeric(nt)
  C[1] <- 1 / N

  # Generation interval weights (with dt baked in for the convolution)
  g <- dgamma(times, shape = popshape, rate = r) * dt
  # Index case kernel (no dt — it's a single person, not a rate)
  g_index <- dgamma(times, shape = popshape, rate = r) / N

  for (i in 2:nt) {
    S <- 1 - C[i-1]
    # Index case + renewal convolution
    j[i] <- S * R0 * (g_index[i] + sum(g[2:i] * j[(i-1):1]))
    C[i] <- C[i-1] + j[i] * dt
  }

  tibble(t = times, j = j, cuminf = C, S = 1 - C)
}

This is equivalent to setting j[1] = 1/(N*dt) (the discretised delta function), but makes the reasoning explicit.

Why it looks reversed with 100 simulations

The milestone time has substantial stochastic spread. For a Gamma\((k, k)\) distribution of \(W\), the variance of \(\log W\) is \(\psi_1(k)\) (the trigamma function), so:

$$\text{SD}(T_{100}) \;\approx\; \frac{\sqrt{\psi_1(k)}}{\alpha}$$

This gives SD \(\approx 2.2\) days for \(\kappa = 0\) and \(\approx 1.8\) days for \(\kappa = 0.25\). With \(n_{\text{sim}} = 100\), the standard error of the median is:

$$\text{SE}_{\text{median}} \approx \frac{\sqrt{\pi/2}}{\sqrt{n}} \cdot \text{SD} \;\approx\; 0.25 \text{ days}$$

So the SE of the difference in medians is \(\approx 0.35\) days. The true difference (0.15 days) is only 0.4 standard errors — well within noise. The apparent reversal between \(\kappa = 0\) and \(\kappa = 0.25\) is a simulation artefact that will disappear with more runs.

The crossing mechanism is real, though

Your intuition about the variance effect is correct — the \(\kappa=0\) and \(\kappa=0.25\) survival curves genuinely cross (at \(t \approx 3.5\) and \(t \approx 7.8\)). In the window between these crossings, \(\kappa=0\) has lower survival (appears faster) because its heavy right tail of \(W\) produces a handful of very fast epidemics. But this crossing isn't strong enough to make \(\kappa=0.25\) the slowest overall — it just means the two curves interleave.

The underlying reason the differences are so small: \(\alpha_{\text{pop}} = 10\) makes the generation interval quite tightly concentrated (CV = 0.32), which limits how much the punctuation parameter \(\kappa\) can spread out \(W\). With a less concentrated generation interval (smaller \(\alpha_{\text{pop}}\)), the separation between \(\kappa\) values would be much more dramatic.

Moment-matching the SEIR generation interval

In the SEIR ODE, a person infected at time 0 spends \(\text{Exp}(\nu)\) time latent, then transmits at constant rate \(\beta\) for an \(\text{Exp}(\gamma)\) infectious period. The population-level infectiousness profile is:

$$A(\tau) = \beta \cdot P(\text{in } I \text{ at time } \tau) = \beta \cdot \frac{\nu}{\nu - \gamma}\left(e^{-\gamma\tau} - e^{-\nu\tau}\right)$$

with \(\nu = 1/2,\ \gamma = 1/3\). Since \(\int_0^\infty A(\tau)\,d\tau = R_0\), the generation interval density is:

$$g(\tau) = \frac{A(\tau)}{R_0} = e^{-\tau/3} - e^{-\tau/2}$$

This is a hypoexponential density — the distribution of \(T_E + T_I\) where \(T_E \sim \text{Exp}(1/2)\) and \(T_I \sim \text{Exp}(1/3)\) are independent. Its moments:

$$E[\tau] = 2 + 3 = 5, \qquad \text{Var}(\tau) = 4 + 9 = 13, \qquad \text{CV} = \frac{\sqrt{13}}{5} \approx 0.72$$

Gamma moment-match

Matching \(\text{Gamma}(\alpha_{\text{pop}},\, r)\) to these two moments: mean \(= \alpha_{\text{pop}}/r = 5\), variance \(= \alpha_{\text{pop}}/r^2 = 13\). Solving:

$$\alpha_{\text{pop}} = \frac{(\text{mean})^2}{\text{var}} = \frac{25}{13} \approx 1.92, \qquad r = \frac{\text{mean}}{\text{var}} = \frac{5}{13} \approx 0.385$$

Why does κ=0.25 look so similar to the spike (κ=0)? [Measles parameters]

Var(W) is highly convex in κ

Your thinking is right that \(\kappa = 0.25\) makes the individual infectiousness profile's SD equal to half the population GI's SD (\(\sqrt{\kappa} = 0.5\)). But the effect on Var(\(W\)) is highly nonlinear — the extra variance beyond the smooth baseline drops like \(R_0^{-2\kappa}\), which is convex and front-loaded.

The key formula

The extra variance beyond the smooth limit decomposes as:

$$\Delta(\kappa) \;=\; \text{Var}(W)\big|_\kappa - \text{Var}(W)\big|_{\kappa=1} \;=\; \frac{R_0^2}{1 - m_2}\;\underbrace{\rho^{2\kappa\alpha_{\text{pop}}}}_{\text{prefactor}} \cdot \underbrace{\left(\rho_2^{(1-\kappa)\alpha_{\text{pop}}} - \rho^{2(1-\kappa)\alpha_{\text{pop}}}\right)}_{\text{bracket}}$$

Since \(\rho^{\alpha_{\text{pop}}} = 1/R_0\), the prefactor simplifies to:

$$\rho^{2\kappa\alpha_{\text{pop}}} = R_0^{-2\kappa}$$

This factor is exponential in \(\kappa\) and drops very steeply near \(\kappa = 0\):

At \(\kappa = 0.25\), the individual profile has half the SD of the population GI — but 71% of the spike's extra variance in \(W\) is retained. You'd need roughly \(\kappa \approx 0.5\) for a halfway effect.

Intuition

The \(R_0^{-2\kappa}\) prefactor comes from the Laplace transform of the jitter component evaluated at \(2\alpha\). The jitter's shape is \(\kappa \alpha_{\text{pop}}\), and even a small shape parameter produces substantial "diversification" — reducing variance by spreading offspring times apart. But the marginal diversification benefit diminishes rapidly: the first 25% of jitter variance eliminates only 29% of the extra W-variance, while the last 25% eliminates 21%.

The effect is stronger at higher \(R_0\) because \(R_0^{-2\kappa}\) drops faster. For \(R_0 = 2\) (influenza), \(R_0^{-0.5} = 0.71\) — a modest drop. For \(R_0 = 12\) (measles), \(R_0^{-0.5} = 0.29\) — a steep drop but still leaving most of the extra variance. The net effect is that \(\kappa = 0.25\) stays close to the spike across all realistic \(R_0\).

Derivation of Var(W) = (cα + σ(1−ψ)α − 1) / (1 − cα)

Deriving the variance of the CMJ limit W

Setup

Each infector produces \(N \sim \text{Poisson}(R_0)\) offspring. Offspring \(j\) is infected at time \(\tau_j = s + \varepsilon_j\), where:

  • \(s \sim \text{Gamma}((1-\psi)\alpha,\, r)\) — shared shift, common to all offspring of this infector
  • \(\varepsilon_j \sim \text{Gamma}(\psi\alpha,\, r)\) — independent jitter, iid across offspring

Marginally, \(\tau_j \sim \text{Gamma}(\alpha, r)\) (by the additivity of Gamma distributions with the same rate).

The Malthusian parameter \(\alpha_M\) satisfies the Euler–Lotka equation:

$$R_0 \cdot E[e^{-\alpha_M \tau}] = 1$$

Define \(\rho = \frac{r}{r + \alpha_M}\), so \(E[e^{-\alpha_M \tau}] = \rho^\alpha\) and the Euler–Lotka equation becomes \(R_0 \rho^\alpha = 1\), i.e. \(\rho^\alpha = 1/R_0\).

Step 1: The branching recursion for W

The CMJ limit \(W = \lim_{t \to \infty} e^{-\alpha_M t} I_b(t)\) satisfies the distributional fixed-point equation:

$$W \stackrel{d}{=} \sum_{j=1}^{N} e^{-\alpha_M \tau_j} W_j$$

where the \(W_j\) are iid copies of \(W\), independent of \((N, s, \varepsilon_1, \ldots)\). Define the discounted offspring sum:

$$S = \sum_{j=1}^{N} e^{-\alpha_M \tau_j}$$

By the Euler–Lotka equation, \(E[S] = R_0 \cdot E[e^{-\alpha_M \tau}] = 1\), and from the fixed point, \(E[W] = E[S] \cdot E[W] = 1\) (starting from \(I_0 = 1\)).

Step 2: Var(W) in terms of Var(S)

From the fixed-point equation, compute \(E[W^2]\) by conditioning on \((N, s, \varepsilon_1, \ldots, \varepsilon_N)\):

$$E[W^2 \mid N, s, \boldsymbol{\varepsilon}] = \sum_j e^{-2\alpha_M \tau_j} E[W^2] + \sum_{j \ne k} e^{-\alpha_M(\tau_j + \tau_k)}$$

Recognising that \(S^2 = \sum_j e^{-2\alpha_M \tau_j} + \sum_{j \ne k} e^{-\alpha_M(\tau_j + \tau_k)}\), this becomes:

$$E[W^2 \mid N, s, \boldsymbol{\varepsilon}] = (E[W^2] - 1) \sum_j e^{-2\alpha_M \tau_j} + S^2$$

Taking expectations and defining \(m_2 = E\!\left[\sum_j e^{-2\alpha_M \tau_j}\right] = R_0 \rho_2^\alpha\) where \(\rho_2 = \frac{r}{r + 2\alpha_M}\):

$$E[W^2] = (E[W^2] - 1)\, m_2 + E[S^2]$$

Rearranging, and using \(E[W] = E[S] = 1\):

$$\boxed{\text{Var}(W) = \frac{\text{Var}(S)}{1 - m_2}}$$

This is valid whenever \(m_2 < 1\), which holds in the supercritical regime.

Step 3: Computing E[S²] by conditioning on (N, s)

Factor the offspring times as \(\tau_j = s + \varepsilon_j\):

$$S = e^{-\alpha_M s} \sum_{j=1}^{N} e^{-\alpha_M \varepsilon_j}$$ $$S^2 = e^{-2\alpha_M s} \left(\sum_{j=1}^{N} e^{-\alpha_M \varepsilon_j}\right)^2$$

Condition on \(N\) and \(s\). The \(\varepsilon_j\) are iid \(\text{Gamma}(\psi\alpha, r)\), so:

\begin{align} E[e^{-\alpha_M \varepsilon}] &= \rho^{\psi\alpha} \\ E[e^{-2\alpha_M \varepsilon}] &= \rho_2^{\psi\alpha} \end{align}

Expanding the square:

$$E\!\left[\left(\sum_j e^{-\alpha_M \varepsilon_j}\right)^2 \,\middle|\, N\right] = N\, \rho_2^{\psi\alpha} + N(N-1)\, \rho^{2\psi\alpha}$$

Taking expectation over \(N \sim \text{Poisson}(R_0)\) (using \(E[N] = R_0\), \(E[N(N-1)] = R_0^2\)):

$$E\!\left[\left(\sum_j e^{-\alpha_M \varepsilon_j}\right)^2\right] = R_0\, \rho_2^{\psi\alpha} + R_0^2\, \rho^{2\psi\alpha}$$

Now take expectation over \(s \sim \text{Gamma}((1-\psi)\alpha, r)\), using \(E[e^{-2\alpha_M s}] = \rho_2^{(1-\psi)\alpha}\):

$$E[S^2] = \rho_2^{(1-\psi)\alpha}\!\left[R_0\, \rho_2^{\psi\alpha} + R_0^2\, \rho^{2\psi\alpha}\right] = R_0\, \rho_2^\alpha + R_0^2\, \rho^{2\psi\alpha}\, \rho_2^{(1-\psi)\alpha}$$

Since \(E[S] = 1\):

$$\text{Var}(S) = R_0\, \rho_2^\alpha + R_0^2\, \rho^{2\psi\alpha}\, \rho_2^{(1-\psi)\alpha} - 1$$

Step 4: Introducing c and σ

Define \(\mu = R_0^{1/\alpha}\), the per-stage growth factor. From \(\rho^\alpha = 1/R_0\), we get \(\rho = 1/\mu\), so \(\alpha_M = (\mu - 1)r\). Then:

$$\rho_2 = \frac{r}{r + 2\alpha_M} = \frac{r}{(2\mu - 1)r} = \frac{1}{2\mu - 1}$$

Define two key ratios:

$$c = \frac{\rho_2}{\rho} = \frac{\mu}{2\mu - 1}, \qquad \sigma = \frac{\rho_2}{\rho^2} = \frac{\mu^2}{2\mu - 1} = \mu \cdot c$$

Both depend only on \(\mu = R_0^{1/\alpha}\), not on \(\psi\).

Express the terms:

First term: \(m_2 = R_0\, \rho_2^\alpha = \rho_2^\alpha / \rho^\alpha = (\rho_2/\rho)^\alpha = c^\alpha\).

Second term: Using \(\rho^{2\psi\alpha} = (\rho^\alpha)^{2\psi} = R_0^{-2\psi}\):

$$R_0^2\, \rho^{2\psi\alpha}\, \rho_2^{(1-\psi)\alpha} = R_0^{2(1-\psi)}\, \rho_2^{(1-\psi)\alpha} = \left(\frac{\rho_2}{\rho^2}\right)^{(1-\psi)\alpha} = \sigma^{(1-\psi)\alpha}$$

where we used \(\rho^{2(1-\psi)\alpha} = R_0^{-2(1-\psi)}\) to write \(R_0^{2(1-\psi)} \rho_2^{(1-\psi)\alpha} = (\rho_2/\rho^2)^{(1-\psi)\alpha}\).

Step 5: Final result

Substituting into \(\text{Var}(W) = \text{Var}(S)/(1 - m_2)\):

$$\boxed{\text{Var}(W) = \frac{c^\alpha + \sigma^{(1-\psi)\alpha} - 1}{1 - c^\alpha}}$$

where \(c = \frac{\mu}{2\mu - 1}\), \(\sigma = \frac{\mu^2}{2\mu - 1}\), \(\mu = R_0^{1/\alpha}\).

Key observations

  • Smooth limit (\(\psi = 1\)): \(\sigma^0 = 1\), so \(\text{Var}(W) = c^\alpha/(1 - c^\alpha) = m_2/(1 - m_2)\). This is the minimum variance — each offspring time is fully independent.
  • Spike limit (\(\psi = 0\)): \(\sigma^\alpha = R_0^2 \rho_2^\alpha\), giving \(\text{Var}(W) = (c^\alpha + \sigma^\alpha - 1)/(1 - c^\alpha)\). This is the maximum — all offspring share the same timing.
  • \(\psi\) only enters through \(\sigma^{(1-\psi)\alpha}\), confirming that narrower profiles (smaller \(\psi\)) always increase Var(\(W\)).
  • Since \(E[W] = 1\), we have \(\text{CV}^2(W) = \text{Var}(W)\).
Conditional moments, Gamma approximation, and time-shift mode

From Var(W) to the mode of the time shift

Conditional moments of W

The random variable \(W\) has a point mass at 0 (extinction, probability \(q\)) and a continuous part on \((0, \infty)\) (survival, probability \(1 - q\)). Since all unconditional moments of \(W\) come entirely from the continuous part:

$$E[W^n] = (1 - q)\, E[W^n \mid W > 0]$$

so the conditional (on survival) raw moments are simply:

$$E[W^n \mid \text{surv}] = \frac{E[W^n]}{1 - q}$$

For the extinction probability, \(q\) is the smallest root of \(q = e^{R_0(q - 1)}\) (Poisson offspring). For \(R_0 \gg 1\), \(q \approx 0\).

First two unconditional moments

We already derived \(E[W] = 1\) and:

$$E[W^2] = 1 + \text{Var}(W) = 1 + \frac{c^\alpha + \sigma^{(1-\psi)\alpha} - 1}{1 - c^\alpha} = \frac{\sigma^{(1-\psi)\alpha}}{1 - c^\alpha}$$

where we combined terms: \(1 + (c^\alpha + \sigma^{(1-\psi)\alpha} - 1)/(1 - c^\alpha) = (1 - c^\alpha + c^\alpha + \sigma^{(1-\psi)\alpha} - 1)/(1 - c^\alpha)\).

Conditional CV²

The conditional coefficient of variation squared is:

\begin{align} \text{CV}^2_{\text{cond}} &= \frac{\text{Var}(W \mid \text{surv})}{(E[W \mid \text{surv}])^2} \\[6pt] &= \frac{E[W^2 \mid \text{surv}]}{(E[W \mid \text{surv}])^2} - 1 \\[6pt] &= \frac{E[W^2]/(1-q)}{1/(1-q)^2} - 1 \\[6pt] &= (1 - q)\, E[W^2] - 1 \end{align}

Substituting \(E[W^2] = \sigma^{(1-\psi)\alpha}/(1 - c^\alpha)\):

$$\text{CV}^2_{\text{cond}} = \frac{(1 - q)\, \sigma^{(1-\psi)\alpha}}{1 - c^\alpha} - 1$$

When \(q \approx 0\): \(\text{CV}^2_{\text{cond}} \approx \text{Var}(W)\), since \(E[W] = 1\) makes \(\text{CV}^2 = \text{Var}\) unconditionally.

Two-moment Gamma approximation

Approximate \(W \mid \text{surv}\) by \(\text{Gamma}(k, \lambda)\), matching the first two moments. Write \(V = \text{CV}^2_{\text{cond}}\) and \(p = 1 - q\) for brevity.

Mean \(k/\lambda = 1/p\) \(k = 1/V\)
CV² \(1/k = V\) \(\lambda = p/V\)

First three moments of the Gamma approximation

For \(\text{Gamma}(k, \lambda)\), the \(n\)-th raw moment is \(\prod_{j=0}^{n-1}(k+j)/\lambda^n\). Using \(k = 1/V\) and \(\lambda = p/V\):

\begin{align} E[W \mid \text{surv}] &= \frac{1}{p} \\[8pt] E[W^2 \mid \text{surv}] &= \frac{1 + V}{p^2} \\[8pt] E[W^3 \mid \text{surv}] &= \frac{(1 + V)(1 + 2V)}{p^3} \end{align}

The first two are exact (matched by construction). The third is a prediction of the Gamma approximation — it will differ from the true \(E[W^3 \mid \text{surv}]\) unless \(W \mid \text{surv}\) happens to be exactly Gamma.

Expanding \(V\) gives the fully explicit forms. For instance:

$$E[W^2 \mid \text{surv}] = \frac{1}{p^2}\left(1 + \frac{p\, \sigma^{(1-\psi)\alpha}}{1 - c^\alpha} - 1\right) + \frac{1}{p^2} = \frac{\sigma^{(1-\psi)\alpha}}{p(1 - c^\alpha)}$$

which matches the exact value \(E[W^2]/(1-q) = \sigma^{(1-\psi)\alpha}/[p(1 - c^\alpha)]\). ✓

Mode of the time shift

The time shift is \(\varsigma = \log(W)/\alpha_M\) (written as \(\log(W)/r\) in the writeup, where \(r\) denotes the growth rate \(\alpha_M\)). Its mode is determined by the mode of \(W \mid \text{surv}\).

The mode of a \(\text{Gamma}(k, \lambda)\) is \((k-1)/\lambda\) (for \(k \geq 1\)). Substituting:

$$\text{mode}(W \mid \text{surv}) = \frac{k - 1}{\lambda} = \frac{1/V - 1}{p/V} = \frac{1 - V}{p}$$

Taking the log and dividing by \(\alpha_M\):

$$\text{mode}_\varsigma = \frac{1}{\alpha_M}\left[\log(1 - V) - \log p\right] = \frac{1}{\alpha_M}\left[\log(1 - \text{CV}^2_{\text{cond}}) - \log(1 - q)\right]$$

When extinction is negligible (\(q \approx 0\), so \(p \approx 1\)):

$$\boxed{\text{mode}_\varsigma \approx \frac{1}{\alpha_M}\log\!\left(1 - \text{CV}^2_{\text{cond}}\right)}$$

which is exactly the expression in the writeup (with the identification \(r \equiv \alpha_M\)).

Interpretation

  • Sign: Since \(V = \text{CV}^2_{\text{cond}} > 0\), the mode is \(\log(1 - V) < 0\), hence \(\text{mode}_\varsigma < 0\). The most likely epidemic is delayed relative to the deterministic trajectory — the mode of \(W\) is less than its mean.
  • Magnitude: Larger \(V\) (more punctuated profile, smaller \(\psi\)) pushes the mode further negative — the typical epidemic starts later. In the extreme, \(V \to 1\) makes \(\text{mode}_\varsigma \to -\infty\).
  • ψ-dependence: Substituting \(V \approx \text{Var}(W) = (c^\alpha + \sigma^{(1-\psi)\alpha} - 1)/(1 - c^\alpha)\), the mode depends on \(\psi\) solely through \(\sigma^{(1-\psi)\alpha}\). The \(\psi = 1\) (smooth) limit gives the least negative mode; \(\psi = 0\) (spike) gives the most negative.
  • Skewness: The Gamma approximation has skewness \(2/\sqrt{k} = 2\sqrt{V}\). More punctuated profiles increase \(V\), making \(W\) more right-skewed: a few epidemics start very early, but the bulk are delayed.

Validity constraint

The mode formula requires \(k \geq 1\), i.e. \(V \leq 1\), i.e. \(\text{CV}^2_{\text{cond}} \leq 1\). When \(V > 1\), the Gamma density is monotonically decreasing (mode at 0), and the mode of the time shift is \(-\infty\) — the most likely outcome becomes extinction or near-extinction. This occurs for very punctuated profiles with moderate \(R_0\). For the spike limit (\(\psi = 0\)) with \(q \approx 0\):

$$V_{\text{spike}} = \frac{c^\alpha + \sigma^\alpha - 1}{1 - c^\alpha} = \frac{c^\alpha + R_0^2 \rho_2^\alpha - 1}{1 - c^\alpha}$$

This exceeds 1 when \(\sigma^\alpha > 2(1 - c^\alpha)\), which can happen for small \(R_0\) or small \(\alpha\).

Overdispersion from periodic contacts × punctuated infectiousness

Deriving the dispersion parameter k

Setup

Consider a population with periodic contact rate:

$$c(t) = \bar{c} + A\sin(\omega t)$$

where \(\bar{c}\) is the mean contact rate, \(A < \bar{c}\) is the amplitude, and \(\omega = 2\pi/P\) is the angular frequency (period \(P\)). An individual infected at calendar time \(t_0\) has a biological infectiousness profile \(a_i(\tau)\) — a density over time-since-infection \(\tau\). Their individual reproduction number is:

$$\mathcal{R}_i = R_0 \int_0^\infty a_i(\tau)\,\frac{c(t_0 + \tau)}{\bar{c}}\, d\tau$$

In the punctuated model, \(a_i(\tau)\) is the density of \(s_i + \varepsilon\) where \(\varepsilon \sim \text{Gamma}(\psi\alpha, r)\), centred on the individual's shared shift \(s_i\). The jitter \(\varepsilon\) spreads the profile, while \(s_i\) locates it in time.

Step 1: Characteristic function argument

Substitute \(c(t_0 + \tau)/\bar{c} = 1 + (A/\bar{c})\sin(\omega(t_0 + \tau))\):

$$\mathcal{R}_i = R_0\!\left[1 + \frac{A}{\bar{c}}\int_0^\infty a_i(\tau)\sin(\omega(t_0 + \tau))\,d\tau\right]$$

The integral is the imaginary part of the characteristic function of \(a_i\) at frequency \(\omega\). Since \(a_i\) is the density of \(s_i + \varepsilon\):

$$\int a_i(\tau)\, e^{i\omega\tau}\, d\tau = e^{i\omega s_i} \cdot \underbrace{\left(\frac{r}{r - i\omega}\right)^{\psi\alpha}}_{\text{char. fn. of } \varepsilon}$$

Write the Gamma characteristic function in polar form:

$$\left(\frac{r}{r - i\omega}\right)^{\psi\alpha} = D\, e^{i\varphi}$$

where the damping factor is:

$$D = \left|\frac{r}{r - i\omega}\right|^{\psi\alpha} = \left(1 + \frac{\omega^2}{r^2}\right)^{-\psi\alpha/2}$$

and \(\varphi = \psi\alpha\cdot\arctan(\omega/r)\) is a phase shift. Then:

$$\int a_i(\tau)\sin(\omega(t_0 + \tau))\,d\tau = D\cdot\sin(\omega(t_0 + s_i) + \varphi)$$

So each individual's reproduction number is:

$$\mathcal{R}_i = R_0\!\left[1 + \frac{A}{\bar{c}}\, D\,\sin\!\big(\omega(t_0 + s_i) + \varphi\big)\right]$$

Step 2: Variance of \(\mathcal{R}_i\)

The phase \(\omega(t_0 + s_i) + \varphi\) varies across individuals (through both infection time \(t_0\) and shared shift \(s_i\)). Averaging over the population — either because \(t_0\) is uniformly distributed over the cycle, or because \(s_i\) has enough spread — \(\sin^2\) averages to 1/2:

$$E[\mathcal{R}_i] = R_0, \qquad \text{Var}(\mathcal{R}_i) = R_0^2 \cdot \frac{A^2}{\bar{c}^2} \cdot D^2 \cdot \frac{1}{2}$$

Step 3: The dispersion parameter

Each individual produces \(N_i \sim \text{Poisson}(\mathcal{R}_i)\) offspring, so:

$$\text{Var}(N_i) = E[\mathcal{R}_i] + \text{Var}(\mathcal{R}_i) = R_0 + \frac{R_0^2 A^2 D^2}{2\bar{c}^2}$$

Matching to the negative binomial mean-variance relation \(\text{Var} = \mu + \mu^2/k\):

$$k = \frac{R_0^2}{\text{Var}(\mathcal{R}_i)} = \frac{2\bar{c}^2}{A^2 D^2}$$

Substituting \(D^2 = (1 + \omega^2/r^2)^{-\psi\alpha}\):

$$\boxed{k = \frac{2\bar{c}^2}{A^2}\left(1 + \frac{\omega^2}{r^2}\right)^{\psi\alpha}}$$

Here \(r = \alpha/T\) is the Gamma rate parameter of the generation interval (not the epidemic growth rate).

Note: The expression in uncontrolled.tex line 37 has a LaTeX typo: \frac{2}{\bar{c}^2}{A^2} renders as \(2A^2/\bar{c}^2\), but the correct coefficient is \(2\bar{c}^2/A^2\). (Sanity check: \(k \to \infty\) as \(A \to 0\), recovering Poisson.)

Step 4: The critical period

Define the critical period:

$$z^*_{\text{per}} = \frac{2\pi}{r} = \frac{2\pi T}{\alpha}$$

This is the period of a contact cycle whose angular frequency matches the Gamma rate \(r\). Since \(\omega = 2\pi/z_{\text{per}}\) where \(z_{\text{per}} = P\) is the contact period:

$$\frac{\omega}{r} = \frac{z^*_{\text{per}}}{z_{\text{per}}}$$

and the formula becomes:

$$k = \frac{2\bar{c}^2}{A^2}\left(1 + \left(\frac{z^*_{\text{per}}}{z_{\text{per}}}\right)^{\!2}\;\right)^{\psi\alpha}$$

Three regimes

Regime Condition \(k\) Interpretation
Slow cycle \(z_{\text{per}} \gg z^*_{\text{per}}\) \(\approx 2\bar{c}^2/A^2\) Contact cycle so slow that even jittered offspring see a nearly constant \(c\). Overdispersion maximal, independent of \(\psi\).
Critical \(z_{\text{per}} \approx z^*_{\text{per}}\) \(\approx (2\bar{c}^2/A^2)\cdot 2^{\psi\alpha}\) Maximum sensitivity to \(\psi\): smooth profiles average out fluctuations, punctuated profiles don't.
Fast cycle \(z_{\text{per}} \ll z^*_{\text{per}}\) \(\sim (z^*_{\text{per}}/z_{\text{per}})^{2\psi\alpha} \to \infty\) Offspring average over many cycles → Poisson. Unless \(\psi = 0\) (spike), where \(k = 2\bar{c}^2/A^2\) always.

Physical intuition

The key insight is that the shared shift \(s\) does not reduce overdispersion — it merely relocates the burst to a different phase of the contact cycle, but all offspring still see the same contact rate. Only the jitter \(\varepsilon\) (controlled by \(\psi\)) averages over the contact fluctuations, because it spreads offspring across different phases.

  • Spike (\(\psi = 0\)): No jitter, \(D = 1\). All offspring hit the contact cycle at the same phase. \(k = 2\bar{c}^2/A^2\), maximal overdispersion regardless of period.
  • Smooth (\(\psi = 1\)): Full jitter with shape \(\alpha\). \(D = (1 + \omega^2/r^2)^{-\alpha/2}\), strong damping of contact fluctuations. \(k\) grows as \((z^*_{\text{per}}/z_{\text{per}})^{2\alpha}\) for fast cycles.
  • Intermediate \(\psi\): Jitter shape is \(\psi\alpha\), giving partial damping. The exponent \(\psi\alpha\) interpolates between the two limits.

Numerical examples

For measles (\(\alpha = 11.36\), \(T = 12.2\) days): \(z^*_{\text{per}} = 2\pi \cdot 12.2/11.36 \approx 6.7\) days.

For influenza (\(\alpha = 2.32\), \(T = 3.2\) days): \(z^*_{\text{per}} = 2\pi \cdot 3.2/2.32 \approx 8.7\) days.

Both are near one week — so weekly contact patterns sit squarely in the critical regime where the punctuation parameter \(\psi\) has maximal effect on overdispersion.

Detailed summary of Middleton & Larremore (2024), "Modeling the transmission mitigation impact of testing for infectious diseases," Science Advances 10, eadk5108.

Middleton & Larremore (2024): Mathematical framework

1. Core quantity: Testing Effectiveness (TE)

The central quantity is Testing Effectiveness (TE), defined in direct analogy to vaccine effectiveness (VE):

$$TE = 1 - \frac{R_{\text{testing}}}{R_{\text{no testing}}} \qquad \Longleftrightarrow \qquad R_{\text{testing}} = (1 - TE)\, R_{\text{no testing}} \tag{1}$$

TE is the proportion by which a testing program reduces population-level transmission, in expectation. It incorporates the test itself, its administration strategy, human behaviors (compliance, participation), and the interplay with viral kinetics. They note this mirrors VE: the expected reduction in post-infection transmission risk for tested vs. untested populations.

2. Infectiousness curve \(\beta(t)\) and the reproduction number

Each individual has a time-varying infectiousness curve \(\beta(t)\), where \(t\) is time since exposure. The individual reproduction number (in the absence of testing) is:

$$\nu_0 = \int_0^\infty \beta(t)\, dt \tag{2}$$

The population-level basic reproduction number is the mean across all individuals:

$$R_{\text{no testing}} = \mathbb{E}[\nu_0] = \int_0^\infty \mathbb{E}[\beta(t)]\, dt \tag{3}$$

The key modeling assumption is that infectiousness is proportional to the logarithm of pathogen load above an empirically established threshold. That is, \(\beta(t) \propto \max(0,\, \log_{10} V(t) - \log_{10} V_{\text{thresh}})\), where \(V(t)\) is the viral load at time \(t\). This is consistent with observations that higher viral loads are associated with more efficient transmission for SARS, SARS-CoV-2, influenza, and RSV.

3. Viral kinetics model

Middleton & Larremore use a piecewise linear model of log pathogen load (a tent function on the log scale), specified by four parameters per infection trajectory:

  • Latent period: \(t_0\) to \(t_0 + \text{LLOQ}\) -- time from exposure to when the virus first exceeds the lower limit of quantification. During this time, \(D(t) = 0\) and \(\beta(t) = 0\).
  • Proliferation phase: Exponential growth (linear increase in log VL) at some proliferation rate, from first detectability to peak VL at time \(t_{\text{peak}}\).
  • Peak viral load: \(V_{\text{peak}}\), drawn from a distribution (Lognormal).
  • Clearance phase: Exponential decline (linear decrease in log VL) from peak back down to undetectable.

All four control points \((t_0, \text{LLOQ})\), \((t_{\text{peak}}, V_{\text{peak}})\), and \((t_f, \text{LLOQ})\) are drawn from distributions (Lognormal with parameters \(\mu, \sigma\)) estimated from the literature for each pathogen (RSV, influenza A, SARS-CoV-2 founder/naive and omicron/experienced). This yields stochastic, individual-level VL trajectories.

Symptom onset time

Symptom onset \(t_{\text{Sx}}\) is drawn relative to peak pathogen load. For example, for founder-strain SARS-CoV-2 in naive hosts, symptoms occur 0 to 3 days after peak VL; for omicron in experienced hosts, 1 to 5 days before peak VL. This reflects the typical manner of reporting in the literature. Asymptomatic infections correspond to trajectories with no symptom onset time drawn.

4. Isolation after diagnosis

If an individual receives a diagnosis at time \(t_{\text{Dx}}\), they isolate and thereby attenuate transmission. For perfect isolation of fixed duration \(\ell\), the individual exits isolation at \(t_{\text{exit}} = t_{\text{Dx}} + \ell\). The post-testing reproduction number becomes:

$$\nu_{\text{testing}}(t_{\text{Dx}}, t_{\text{exit}}) = \int_0^{t_{\text{Dx}}} \beta(t)\, dt + \int_{t_{\text{exit}}}^{\infty} \beta(t)\, dt \tag{5}$$

This has the intuitive interpretation: transmission that occurred before diagnosis plus transmission that occurs after exiting isolation. Transmission during isolation is assumed to be zero (perfect isolation). They note that imperfect/partial isolation is developed in Supplementary Materials, where time-varying impacts of behavioral changes (masking, increased ventilation) can be modeled instead of binary on/off isolation.

5. Diagnosis time as a random variable

The time of diagnosis \(t_{\text{Dx}}\) depends on test availability, schedule, symptom timing, and compliance. It is modeled as a random variable with (improper) probability density \(f(t_{\text{Dx}})\). This leads to an effective infectiousness in the testing scenario, averaged over possible diagnosis times:

$$\bar{\nu}_{\text{testing}} = \int_0^\infty \nu_{\text{testing}}(t_{\text{Dx}})\, f(t_{\text{Dx}})\, dt_{\text{Dx}} \tag{6}$$

Substituting and rearranging using the cumulative probability of diagnosis \(F(t)\):

$$\bar{\nu}_{\text{testing}} = \nu_0 - \int_0^\infty \beta(t)\!\left[F(t) - F(t - \ell)\right]\, dt \tag{7}$$

The term \(F(t) - F(t - \ell)\) is the probability of being in isolation at time \(t\): diagnosed before \(t\) but not yet released (released \(\ell\) days after diagnosis). Note that \(F(t)\) is an improper CDF -- it need not reach 1 as \(t \to \infty\), because not all individuals will necessarily be diagnosed.

They observe that if \(F(t)\) is reinterpreted as the CDF of symptom onset (symptoms as a perfect diagnostic test), with exhaustive isolation thereafter (large \(\ell\)), Eq. 7 recovers the core notion of symptom-based controllability from Fraser et al. (2004).

6. Participation and the effective reproduction number

Let \(p\) be the proportion of the population participating in the testing program. Participants have their infectiousness reduced from \(\nu_0\) to \(\bar{\nu}_{\text{testing}}\); non-participants are unaffected. The effective reproduction number is:

$$R_{\text{testing}} = p\,\mathbb{E}[\bar{\nu}_{\text{testing}}] + (1 - p)\,\mathbb{E}[\nu_0] \tag{4}$$

where the expectations are over the heterogeneous within-host kinetics across the population.

7. Calculating the diagnosis CDF \(F(t)\): testing, compliance, detectability, and TAT

The CDF \(F(t)\) -- the probability of having received a positive diagnosis by time \(t\) -- is built from three components:

7a. Test administration rate \(A(t)\)

\(A(t)\) is the prescribed rate of test administration in tests per day. For sufficiently small \(\delta\):

$$\Pr(\text{test administered between } t \text{ and } t + \delta) = \int_t^{t+\delta} A(t')\, dt' \tag{8}$$

Examples: \(A(t) = 1/7\) models weekly screening; \(A(t) = 2\) for \(4 \leq t \leq 6\) represents twice-daily testing in a two-day window.

7b. Detectability indicator \(D(t)\)

\(D(t)\) is a binary indicator: \(D(t) = 1\) when the infection is analytically detectable at time \(t\) (pathogen load above the test's limit of detection, LOD), and \(D(t) = 0\) otherwise. This defines the detectability window.

7c. Cumulative number of possible positive tests

Combining administration with detectability, the cumulative number of scheduled tests with the potential to return a positive result by time \(t\) is:

$$m(t) = \int_0^{t - \text{TAT}} A(t')\, D(t')\, dt' \tag{9}$$

The upper limit is shifted by TAT (turnaround time) -- the delay between a test's administration and the return of actionable results. This is a crucial parameter: even if a test is taken during the detectable window, the result is not available until TAT later.

7d. Compliance and test failure

  • Compliance \(c\): Independent Bernoulli probability that each prescribed test is actually taken.
  • Test failure \(\phi\): Independent Bernoulli probability that a test fails for non-analytical reasons (e.g., poor biospecimen collection).

Combining these, the CDF of having received a positive test by time \(t\) is:

\begin{align} F(t) &= 1 - \left[1 - \overline{m}(t)\right]\left[1 - c(1 - \phi)\right]^{m(t) - \overline{m}(t)} \notag \\ &\quad - \overline{m}(t)\left[1 - c(1 - \phi)\right]^{m(t) - \overline{m}(t) + 1} \tag{10} \end{align}

where \(\overline{m}(t) = m(t) \bmod 1\). This function reflects that for a monotonically increasing number of possible positive tests \(m(t)\), there are two possibilities: one may have taken either \(m(t) - \overline{m}(t)\) tests (with probability \(1 - \overline{m}(t)\)) or \(m(t) - \overline{m}(t) + 1\) tests (with probability \(\overline{m}(t)\)). Each such test may fail with probability \(\phi\) or be missed with probability \(1 - c\).

This is an improper CDF -- it need not reach 1, reflecting that not all individuals will be diagnosed.

7e. Constant-rate special case

When \(A(t)D(t)\) is a constant \(\bar{A}\) between times \(t_1\) and \(t_2\) (and 0 otherwise), the improper PDF simplifies to:

$$f(t) = \bar{A}c(1 - \phi)\left[1 - c(1 - \phi)\right]^{m(t) - \overline{m}(t)} \tag{11}$$

for \(t_1 + \text{TAT} \leq t \leq t_2 + \text{TAT}\), and 0 otherwise.

7f. Elective testing at symptom onset

For elective testing triggered by symptoms at time \(t_{\text{Sx}}\), they model individuals as waiting \(x\) days after symptoms and then testing for \(y\) days at rate \(\bar{A}\) per day, leading to a test administration function:

$$A(t, t_{\text{Sx}}) = \bar{A}\!\left[u(t - t_{\text{Sx}} - x) - u(t - t_{\text{Sx}} - x - y)\right] \tag{12}$$

where \(u(\cdot)\) is the unit step function. The detection CDF then depends on \(t_{\text{Sx}}\) and must be integrated over the \(t_{\text{Sx}}\) distribution (Eq. 13).

8. The TE formula

Substituting Eq. 7 (\(\bar{\nu}_{\text{testing}}\)) into Eq. 4 (\(R_{\text{testing}}\)) and then into Eq. 1 (\(TE\)):

$$TE = p \int_0^\infty \mathbb{E}\!\left\{\beta(t)\!\left[F(t) - F(t - \ell)\right]\right\}\, dt \;\bigg/\; \int_0^\infty \mathbb{E}[\beta(t)]\, dt \tag{14}$$

The numerator is the expected infectiousness averted by isolation; the denominator is the total expected infectiousness without testing. Both integrands are expected infectiousness trajectories -- ensemble averages over the population's heterogeneous within-host kinetics.

When isolation is of sufficient duration to prevent all post-diagnosis transmission (large \(\ell\)), this simplifies to:

$$TE = p \int_0^\infty \mathbb{E}[\beta(t)\, F(t)]\, dt \;\bigg/\; \int_0^\infty \mathbb{E}[\beta(t)]\, dt \tag{15}$$

Note the subtle but important distinction: the numerator involves \(\mathbb{E}[\beta(t)\, F(t)]\), not \(\mathbb{E}[\beta(t)] \cdot \mathbb{E}[F(t)]\). The joint expectation matters because \(\beta(t)\) and \(F(t)\) are correlated -- both depend on the same within-host viral kinetics trajectory.

9. Ascertainment (protocol sensitivity)

The long-time limit of \(F(t)\) gives the probability that an individual is ever diagnosed. Ascertainment \(\alpha\) (also called "protocol sensitivity" in the literature) is the population-level fraction of infections diagnosed:

$$\alpha = p \cdot \lim_{t \to \infty} \mathbb{E}[F(t)] \tag{16}$$

A key empirical finding of the paper is that ascertainment and TE are only weakly related. For instance, for influenza A with elective RDT testing after symptoms, TE = 38% but ascertainment = 60%; for elective RT-qPCR testing, TE dropped by 36 percentage points (to 2%) while ascertainment dropped by only 10 percentage points (to 50%). This is because TE depends on when diagnoses occur relative to the infectiousness curve, not just whether they occur.

10. Mean generation interval and selection coefficient

The mean generation interval can be computed from the centers of mass of the expected infectiousness curves. The integrands in Eq. 14 represent the expected infectiousness trajectories with (numerator) and without (denominator) testing. These estimates may be important for post hoc estimation of TE from empirical data, since methods for estimating \(R\) from case counts often rely on the mean generation interval.

If a single test can diagnose two strains of the same pathogen, testing can alter the selection coefficient. The selection coefficient shifts from \(s = R_A/R_B\) to \(s_{\text{testing}} = s \cdot (1 - TE_A)/(1 - TE_B)\).

11. Test consumption

They estimate two quantities for test consumption:

11a. No-diagnosis case

If no diagnosis is ever made, the expected number of tests consumed is a weighted combination of tests taken during the detectable window (each at rate \(c\phi/(1 - c + c\phi)\), reflecting the probability that a planned test is consumed given absence of diagnosis) and during the non-detectable window (each at rate \(c\)):

$$\bar{q}_{\text{no-Dx}} = \frac{c\phi}{1 - c + c\phi}\int_0^\infty A(t)D(t)\, dt + c \int_0^\infty A(t)[1 - D(t)]\, dt \tag{17}$$

11b. Diagnosis case

For a fixed time of diagnosis \(t_{\text{Dx}}\), the expected tests consumed is 1 (the diagnosing test) plus any tests administered before it:

$$\bar{q}_{\text{Dx}}(t_{\text{Dx}}) = 1 + c\!\left[\left|\int_0^{t_{\text{Dx}} - \text{TAT}} A(t)\, dt\right| + \left|\int_{t_{\text{Dx}} - \text{TAT}}^{t_{\text{Dx}}} A(t)\, dt\right|\right] \tag{18}$$

The total expected test consumption per infected individual is then obtained by combining these, weighted by ascertainment. The overall population test consumption rate (including non-infected individuals) is:

$$Q = p\!\left[(\theta\, se_{\text{ref.}})\,\bar{q}_{\text{infected}} + (1 - \theta)(1 - sp_{\text{ref.}})\,\bar{q}_{\text{not infected}}\right] \tag{22}$$

where \(\theta\) is the prevalence, \(se_{\text{ref.}}\) and \(sp_{\text{ref.}}\) are sensitivity and specificity of the referral scheme (e.g., contact tracing, symptom presence), and \(\bar{q}_{\text{not infected}} = c \int_0^\infty A(t)\, dt\).

12. Days spent in isolation

For fixed-duration perfect isolation of length \(\ell\), the average isolation days per diagnosed individual is simply \(d = \ell \cdot \alpha\). They also model test-to-exit (TTE) strategies, where individuals test daily (using the same or a different test) after a minimum waiting period in isolation, and exit upon a negative test result. This is covered in the Supplementary Materials (Eq. S10).

13. Imperfect isolation and behavioral changes

While the main text develops perfect isolation (zero transmission during isolation), the Supplementary Materials provide equations for partial and time-varying impacts of behavioral changes such as masking or increased ventilation. The general form replaces the binary "in/out of isolation" with a time-varying infectiousness attenuation factor during the isolation period.

14. Parameterization of viral kinetics

The viral kinetics model uses Lognormal distributions for peak VL, proliferation duration, and clearance duration, with parameters \(\mu\) and \(\sigma\) adapted from published means, 95% credible intervals, and sample sizes. Specifically:

  • RSV: Challenge studies (Refs 28, 29). 2-4 day latent, 3-6 day clearance. VL between 4.5 and 8 \(\log_{10}\) cp RNA/ml.
  • Influenza A: Challenge studies (Refs 30-32). 0.5-2 day latent. Peak VL at 1-3 days after symptom onset. VL between 6 and 8.5 \(\log_{10}\) cp RNA/ml.
  • SARS-CoV-2 (founder/naive): Prospective longitudinal studies (Refs 17, 34). Higher peaks, longer detectability windows (~6.6 days RDT detection).
  • SARS-CoV-2 (omicron/experienced): Lower peaks, shorter clearance, shorter detectability windows (~2.2 days RDT detection), earlier symptom onset relative to peak VL.

Because within-host kinetics vary from one infection to another, each trajectory is parameterized using independent draws from random variables for each control point. They use \(10^5\) stochastic realizations for Monte Carlo estimation.

15. Analytical sensitivity and LOD

LODs are drawn from the literature for RT-qPCR and RDT tests for each pathogen. Because of variability across assays, a single LOD per pathogen and test type is used. Test failure rates \(\phi\) were available only for SARS-CoV-2 (~5%); the same rate was assumed for RSV and influenza A to model sample contamination, poor biospecimen collection, or manufacturing error.

16. Key findings and their mathematical basis

  • TE varies considerably by pathogen and strategy: Weekly RDT screening with 50% compliance gave TE of 13% (RSV), 7% (influenza A), and 13% (SARS-CoV-2 omicron). Elective RDT at symptom onset: 21% (RSV), 38% (influenza A), 25% (SARS-CoV-2).
  • Ascertainment \(\neq\) TE: Ascertainment was only weakly correlated with TE across scenarios (11-14% ascertainment for weekly screening vs. up to 60% for elective post-symptom testing, but TE did not track proportionally).
  • RT-qPCR vs RDT reversal: For founder-strain/naive SARS-CoV-2, RDTs provided higher TE than RT-qPCR (replicating earlier literature). For omicron/experienced hosts, this reversed -- RT-qPCR gave higher TE despite its 2-day TAT, due to shortened RDT detectability windows and the larger gap between PCR and RDT detectability.
  • Presymptomatic transmission limits TE: The existence of asymptomatic and presymptomatic transmission implies TE < 1 for symptom-driven testing, regardless of test quality.

17. Relationship to the Fraser et al. (2004) controllability framework

Eq. 7 with \(F(t)\) reinterpreted as the CDF of symptom onset and \(\ell \to \infty\) recovers the Fraser et al. notion of symptom-based controllability. However, Middleton & Larremore generalize this by:

  1. Making \(t_{\text{Dx}}\) depend on the test's properties and within-host kinetics (not just symptom onset)
  2. Including compliance, test failure, and turnaround time
  3. Allowing finite isolation duration
  4. Modeling heterogeneous viral kinetics across individuals

18. Computational approach

All calculations are performed either by integration (when \(A(t)D(t)\) is a constant, leading to closed-form expressions) or by Monte Carlo simulation (\(10^5\) stochastic realizations of viral load trajectories, symptom onset, test timing, and diagnosis). The ensemble mean of the infectiousness curves with and without testing (Fig. 1B) directly yields the TE estimate. They note this is "similar in spirit to the CEPAC model" for HIV, which provides stochastic individual-level realizations of infection dynamics, costs, and outcomes.

Formalizing how ψ enters screening effectiveness — the Gamma convolution invariance

The Gamma convolution invariance and its breaking

The invariance theorem

Theorem. If individual \(i\) is detected at a time \(\tau_{\text{det}}\) that is independent of \((l_i, \varepsilon_j)\) — e.g., at a fixed time after infection — then with perfect immediate isolation (\(\eta = 1\), \(\delta_{\text{act}} = 0\)):

$$E[\theta_i] = S_\alpha(\tau_{\text{det}}) = P(\text{Gamma}(\alpha, \beta) > \tau_{\text{det}})$$

which is independent of \(\psi\).

Proof. Write \(\theta_i = S_\psi(\tau_{\text{det}} - l_i) = P(\varepsilon > \tau_{\text{det}} - l_i)\) where \(\varepsilon \sim \text{Gamma}(\psi\alpha, \beta)\) is independent of \(l_i \sim \text{Gamma}((1-\psi)\alpha, \beta)\). Taking expectations:

$$E[\theta_i] = P(\varepsilon + l_i > \tau_{\text{det}}) = P(\text{Gamma}(\alpha, \beta) > \tau_{\text{det}})$$

by the convolution property \(\text{Gamma}(\psi\alpha, \beta) + \text{Gamma}((1-\psi)\alpha, \beta) \stackrel{d}{=} \text{Gamma}(\alpha, \beta)\). \(\square\)

Interpretation. Since the population-level profile \(A(\tau) = \text{Gamma}(\alpha, \beta)\) is invariant across \(\psi\), any detection mechanism that doesn't "see" the individual's profile shape produces \(\psi\)-independent TE. All \(\psi\)-dependence must come from detection being coupled to individual biology.

How mode-anchored detection breaks invariance

For screening, \(\tau_{\text{det}} \in \mathcal{W}_i = [l_i + m_\psi - d_{\text{pre}},\, l_i + m_\psi + d_{\text{post}}]\). Write \(\tau_{\text{det}} = l_i + m_\psi + V\) where \(V \in [-d_{\text{pre}}, d_{\text{post}}]\). Then \(l_i\) cancels:

$$\theta_i = \eta \cdot S_\psi(m_\psi + V + \delta_{\text{act}})$$

The convolution can't fire because \(l_i\) appears on both sides of the comparison \(\varepsilon > \tau_{\text{det}} - l_i = m_\psi + V + \delta_{\text{act}}\). We're left comparing a \(\text{Gamma}(\psi\alpha, \beta)\) variable to its own mode plus an offset — a fundamentally \(\psi\)-dependent quantity.

Monotonicity of the supra-modal mass

Proposition. Let \(a = \psi\alpha\). Then \(S_\psi(m_\psi) = P(\text{Gamma}(a, \beta) > \text{mode})\) is strictly decreasing from 1 to \(1/2\) as \(a\) increases from 0 to \(\infty\).

Proof sketch (integer \(a\)). By the Gamma–Poisson duality:

$$P(\text{Gamma}(a, 1) > a - 1) = P(\text{Poisson}(a-1) \leq a - 1)$$

This is \(P(\text{Poisson}(\lambda) \leq \lambda)\) with \(\lambda = a - 1\), which is strictly decreasing in \(\lambda\) and converges to \(1/2\) by the CLT. At \(a \leq 1\): mode = 0, and \(S_\psi(0) = 1\).

The overlap rule: a unified TE formulation across all triggers

The overlap rule

Unified formulation

Define the isolation offset \(D = \tau_{\text{iso}} - l_i\) (time from biological onset to isolation) and a transmission quantum \(\varepsilon \sim \text{Gamma}(\psi\alpha, \beta)\), drawn independently from the individual biological profile. Then:

$$\text{TE} = \eta \cdot P(\text{det}) \cdot P(\varepsilon > D)$$

TE is the probability that a random quantum of transmission falls after isolation.

Mode-anchored triggers (screening, symptoms)

For these triggers, \(l_i\) cancels from \(D\), giving \(D = m_\psi + D_0\) where \(D_0\) is \(\psi\)-free. Defining the mode-centred profile \(\tilde{\varepsilon} = \varepsilon - m_\psi\):

$$\text{TE} = \eta \cdot P(\text{det}) \cdot P(\tilde{\varepsilon} > D_0)$$
Trigger\(D_0\) (isolation offset from mode)
Screening\(V + \delta_{\text{act}}\), where \(V \in [-d_{\text{pre}}, d_{\text{post}}]\)
Symptoms\(\delta_{\text{sym}} + \delta_{\text{act}}\)

The pithy rule

As \(\psi \to 0\), the mode-centred profile \(\tilde{\varepsilon}\) concentrates at 0 (all transmission at the peak):

$$\text{TE}_{\text{spike}} \;\approx\; \eta \cdot P(\text{det}) \cdot P(D_0 < 0)$$

Spike profiles convert TE into a binary bet: was isolation before the peak? Smooth profiles give a graded overlap \(E[S_\psi(m_\psi + D_0)]\) that always captures some tail.

  • Spikes win when \(P(D_0 < 0)\) is large (isolation typically before the mode).
  • Smooth wins when \(P(D_0 < 0)\) is small (isolation typically after the mode).

This explains the screening–symptom reversal in one line: screening with large \(d_{\text{pre}}\) puts \(D_0\) mostly negative → spikes benefit. Symptoms with \(\mu_{\text{sym}} > 0\) put \(D_0\) entirely positive → smooth benefits.

Contact tracing: convolution absorption

For a traced contact \(j\), the Gamma convolution absorbs \(l_j\):

$$E_{l_j, \varepsilon_j'}[\theta_j] = \eta \cdot P(\varepsilon_j' + l_j > \tau_{\text{iso}}^{(j)}) = \eta \cdot S_\alpha(\tau_{\text{iso}}^{(j)})$$

replacing \(S_\psi\) with the \(\psi\)-independent population survival function \(S_\alpha\). All \(\psi\)-dependence enters through the isolation timing \(D^{\text{CT}} = \max(0,\, m_\psi + \delta_{\text{sym}}^A + \delta_{\text{trace}} - \varepsilon_j) + \delta_{\text{act}}\), not through the profile. Key assumptions

The overlap rule and its consequences rest on the following assumptions. These are numbered for reference; violations of each are noted.

A1. Shifted Gamma model

The individual infectiousness profile is \(a_i(\tau) = R_0 \cdot f_\psi(\tau - l_i)\) with \(l_i \sim \text{Gamma}((1-\psi)\alpha, \beta)\) and \(f_\psi = \text{Gamma}(\psi\alpha, \beta)\) density. This is the specific functional form that makes \(l_i + \varepsilon \sim \text{Gamma}(\alpha, \beta)\). Without Gamma additivity, the convolution invariance fails, and TE at fixed detection time would depend on \(\psi\).

A2. \(\psi\)-independent detectability window

The half-widths \(d_{\text{pre}}\) and \(d_{\text{post}}\) do not depend on \(\psi\). This is the most debatable assumption. In reality, spikier profiles have higher peak viral loads but shorter durations above the limit of detection, potentially narrowing the detectability window. If the window width \(w\) shrinks with \(\psi\), this reduces \(P(\text{det})\) for spikes, partially or fully offsetting their advantage. The overlap rule still holds formally (with \(\psi\)-dependent \(w\)), but the clean separation \(D_0 = V + \delta_{\text{act}}\) with \(\psi\)-free \(D_0\) would break.

A3. Detection anchored to the mode

The detectability window is centred on \(l_i + m_\psi\). This is what causes \(l_i\) to cancel from the isolation offset \(D\). If detection were anchored to a different feature — e.g., a fixed viral load threshold that doesn't track the mode, or the mean \(l_i + \psi\alpha/\beta\) instead of the mode \(l_i + (\psi\alpha - 1)/\beta\) — the cancellation would differ by an additive \(\psi\)-dependent shift \(1/\beta\). This is a minor effect (for the mode vs. mean distinction, the difference is \(1/\beta\), independent of \(\psi\)), but matters conceptually: the overlap rule requires that the anchor point shifts with \(l_i\) one-for-one.

A4. Independence of detection timing and transmission quanta

The offset \(V\) (position of the first positive test within the window) is independent of \(\varepsilon\) (the individual's own biological jitter). This holds because the screening schedule is external to the individual's profile. Similarly, \(\delta_{\text{sym}}\) is independent of \(\varepsilon\) (symptom timing doesn't depend on which specific quanta of transmission the individual produces). This assumption does not require perfect sensitivity (\(p_{\text{sens}} = 1\)). With imperfect sensitivity, \(V\) is stochastically larger (failed early tests push detection later in the window), but \(V\) remains \(\psi\)-independent. The distribution of \(D_0\) changes with \(p_{\text{sens}}\), but the overlap rule \(\text{TE} = \eta \cdot P(\text{det}) \cdot P(\tilde{\varepsilon} > D_0)\) holds for any \(p_{\text{sens}}\).

A5. Symptom onset anchored to the mode

\(\tau_{\text{sym}} = l_i + m_\psi + \delta_{\text{sym}}\). This is the symptom-trigger analogue of A3. It means symptoms track the individual's peak infectiousness, shifted by a \(\psi\)-independent random variable \(\delta_{\text{sym}}\). If symptoms were instead anchored to infection time (i.e., \(\tau_{\text{sym}} = t_i + \delta_{\text{sym}}'\)), then \(l_i\) would not cancel, the convolution invariance would apply, and TE would be \(\psi\)-independent — all the interesting structure would vanish.

The equivalence point of the overlap rule

Where exactly is the crossover — the isolation offset \(D_0^*\) at which neither spike nor smooth profiles are preferred?

The question

The overlap rule gives the fraction of transmission averted as

$$\frac{\theta}{\eta} = S_\psi(m_\psi + D_0) = \Pr\!\bigl(\varepsilon > m_\psi + D_0\bigr), \qquad \varepsilon \sim \text{Gamma}(\psi\alpha,\,\beta).$$

where \(m_\psi = (\psi\alpha - 1)/\beta\) is the mode. Equivalently, writing \(\tilde\varepsilon = \varepsilon - m_\psi\) for the mode-centred profile:

$$\frac{\theta}{\eta} = \Pr(\tilde\varepsilon > D_0).$$

The crossover \(D_0^*\) is the value at which this probability is independent of \(\psi\) — i.e. the isolation offset at which spikes and smooth profiles avert the same fraction.

Why \(D_0 = 0\) is not the answer

At \(D_0 = 0\), we evaluate \(\Pr(\varepsilon > m_\psi) = \Pr\!\bigl(\text{Gamma}(\psi\alpha, \beta) > \text{mode}\bigr)\). The Gamma density is right-skewed, so more than half the mass lies above the mode. The degree of skewness depends on the shape parameter \(a = \psi\alpha\):

The median–mode gap

The natural 50-50 split point of any distribution is its median, not its mode. If we set \(D_0\) equal to the median–mode gap, then \(\Pr(\tilde\varepsilon > D_0) = \Pr(\varepsilon > \text{median}) = \tfrac{1}{2}\) regardless of shape.

For a \(\text{Gamma}(a, \beta)\) distribution, a classical asymptotic expansion gives:

$$\text{median} \;=\; \frac{a - \tfrac{1}{3} + O(a^{-1})}{\beta}$$

$$\boxed{\;D_0^* \;=\; \text{median} - \text{mode} \;\approx\; \frac{2}{3\beta}\;}$$

with corrections of order \(O(1/(a\beta)) = O(1/(\psi\alpha\beta))\). The key point: this gap is approximately independent of the shape parameter \(a = \psi\alpha\), so it serves as an (approximate) universal crossover.

General crossover theorem: beyond the Gamma

Does the median–mode crossover hold for any biological profile, not just the Gamma? If we used a Normal, would D₀* = 0? If left-skewed, would D₀* < 0?

Setup

Consider any unimodal distribution family \(\{F_\psi\}\) for the individual biological profile \(\varepsilon\), parameterized by a concentration parameter \(\psi\). As \(\psi \to 0\), the profile concentrates into a spike at its mode \(m_\psi\). The overlap rule gives the fraction of transmission averted as

$$\frac{\theta}{\eta} = S_\psi(m_\psi + D_0) \;=\; \Pr(\varepsilon > m_\psi + D_0), \qquad \varepsilon \sim F_\psi.$$

The crossover \(D_0^*\) is the offset where this quantity is independent of \(\psi\).

The theorem

The argument is elementary. For each \(\psi\):

  • \(S_\psi(m_\psi + D_0)\) is a continuous, strictly decreasing function of \(D_0\), dropping from \(\approx 1\) to \(0\).
  • It passes through \(\tfrac{1}{2}\) at exactly \(D_0 = \text{median}_\psi - m_\psi\).

Theorem. If the median–mode gap \(\Delta_\psi := \text{median}_\psi - \text{mode}_\psi\) is approximately independent of \(\psi\), then all survival curves pass through a common point

$$\bigl(D_0^*,\;\tfrac{1}{2}\bigr), \qquad D_0^* = \Delta_\psi \approx \text{const},$$

and this is the universal spike–smooth crossover. The sign of \(D_0^*\) equals the sign of the skewness:

  • Right-skewed (mode < median): \(D_0^* > 0\). Spikes preferred for early isolation.
  • Symmetric (mode = median): \(D_0^* = 0\). Crossover exactly at the peak.
  • Left-skewed (mode > median): \(D_0^* < 0\). Spikes preferred for late isolation.

If \(\Delta_\psi\) depends on \(\psi\), the pairwise crossings scatter over a range of \(D_0\) values and no single clean crossover exists.

Why the Gamma has a clean crossover

For \(\text{Gamma}(a, \beta)\), the mean–mode gap is

$$\text{mean} - \text{mode} = \frac{a}{\beta} - \frac{a-1}{\beta} = \frac{1}{\beta} \qquad \text{(exactly shape-independent).}$$

Pearson's classical relation \(\text{mean} - \text{mode} \approx 3(\text{mean} - \text{median})\) then gives

$$\text{median} - \text{mode} \;\approx\; \tfrac{2}{3}(\text{mean} - \text{mode}) \;=\; \frac{2}{3\beta}$$

with corrections \(O(1/(a\beta))\). The leading term is independent of \(a = \psi\alpha\), giving the clean crossover. Numerically:

Other skewed families (log-normal, Weibull) lack this and have no single crossover.

This is ultimately a consequence of the asymmetry between the rising and falling phases of infectiousness.

Random detection times: variance helps spikes

What if \(D_0\) is random? Is the condition for spike preference simply \(E[D_0] < D_0^*\)?

Setup

When detection timing is uncertain, \(D_0 \sim G\) with mean \(\mu\) and variance \(\sigma^2\), and the expected fraction averted becomes

$$h(\psi) \;=\; E_{D_0}\!\bigl[\tilde S_\psi(D_0)\bigr] \;=\; \int \tilde S_\psi(d)\, g(d)\,\mathrm{d}d \;=\; \Pr(\tilde\varepsilon > D_0)$$

where \(\tilde\varepsilon \perp\!\!\!\perp D_0\). Spikes are preferred when \(h(\psi_{\text{spike}}) > h(\psi_{\text{smooth}})\).

Leading-order answer: yes, \(E[D_0] < D_0^*\)

For small variance, expand about the mean:

$$h(\psi) \;\approx\; \tilde S_\psi(\mu) + \tfrac{\sigma^2}{2}\,\tilde S_\psi''(\mu)$$

The first term gives the deterministic result: spike wins iff \(\mu < D_0^*\). So to leading order, the user's intuition is correct — the mean detection offset must fall before the median–mode gap.

The variance correction: spikes benefit from noise

The second-order term reveals a surprise. Since \(\tilde S'' = -\tilde f'\), and the density \(\tilde f\) is decreasing past the mode (at \(D_0^* > 0\)), we have \(\tilde f'(D_0^*) < 0\) and therefore \(\tilde S''(D_0^*) > 0\) — the survival function is convex in this region.

By Jensen's inequality: \(E[\tilde S(D_0)] > \tilde S(E[D_0])\). Both spike and smooth benefit from this convexity uplift, but the spike benefits far more because its survival function is steeper (more convex):

  • Lucky early detections (\(D_0 \ll 0\)) capture the spike's entire burst — a windfall.
  • Unlucky late detections (\(D_0 \gg 0\)) miss nothing extra — the spike already fired.

The payoff is asymmetric: the upside of early detection (avert everything) exceeds the downside of late detection (miss everything, but you'd have missed most of it anyway). For smooth profiles, the payoff is more linear — early or late detection matters proportionally. This asymmetry is Jensen's inequality at work: the spike's step-like survival function is convex, making it the natural beneficiary of randomness.

General variance–skewness theorem

Does the "variance helps spikes" result hold for arbitrary unimodal profiles, not just the Gamma? Can we make a distribution-free argument?

The argument

Yes — and the argument requires only one fact from elementary calculus: a unimodal density increases before its mode and decreases after.

Consider any unimodal, mode-centred profile \(\tilde\varepsilon\) with density \(\tilde f\) having its mode at \(d = 0\). The survival function \(\tilde S(d) = \Pr(\tilde\varepsilon > d)\) satisfies

$$\tilde S''(d) = -\tilde f'(d) \;=\; \begin{cases} < 0 & d < 0 \;\;\text{(concave: density still rising)} \\[3pt] > 0 & d > 0 \;\;\text{(convex: density falling)} \end{cases}$$

The inflection point is at \(d = 0\) (the mode), and every unimodal distribution has this structure regardless of parametric form.

Step 1: Where does the crossover sit?

The crossover \(D_0^* = \text{median} - \text{mode}\). For right-skewed profiles, \(D_0^* > 0\) — the crossover lies in the convex region. For left-skewed, \(D_0^* < 0\) — the crossover lies in the concave region.

Step 2: Which profile has more curvature?

For a density with scale \(\sigma_\psi\) (standard deviation), dimensional analysis gives

$$\tilde f \sim \frac{1}{\sigma_\psi}, \qquad |\tilde f'| \sim \frac{1}{\sigma_\psi^2}, \qquad |\tilde S''| = |\tilde f'| \sim \frac{1}{\sigma_\psi^2}$$

A more concentrated profile (smaller \(\sigma_\psi\)) has greater curvature \(|\tilde S''|\) at any fixed point near the mode. In particular, at \(D_0^*\):

$$|\tilde S_{\text{spike}}''(D_0^*)| \;\gg\; |\tilde S_{\text{smooth}}''(D_0^*)|$$

Verified numerically: for our Gamma parameters, \(|\tilde S''(D_0^*)| = 2.05\) (spike) vs \(0.04\) (smooth) — a 50-fold ratio.

Step 3: Jensen's inequality decides the sign

Define \(\Delta(d) = \tilde S_{\text{spike}}(d) - \tilde S_{\text{smooth}}(d)\), which crosses zero at \(D_0^*\). By Taylor expansion around \(D_0^*\):

$$\Delta(D_0^* \pm t) \;\approx\; \frac{t^2}{2}\,\Delta''(D_0^*)$$

For \(D_0\) centred at \(D_0^*\) with variance \(\sigma^2\):

$$E\bigl[\Delta(D_0)\bigr] \;\approx\; \frac{\sigma^2}{2}\,\Delta''(D_0^*) \;=\; \frac{\sigma^2}{2}\bigl[\tilde S_{\text{spike}}''(D_0^*) - \tilde S_{\text{smooth}}''(D_0^*)\bigr]$$

Theorem (distribution-free). For any unimodal biological profile, the effect of variance in detection timing on the spike–smooth comparison is determined by the skewness of the profile:

Writing \(D_0^{*,\text{eff}}\) for the effective crossover (the value of \(E[D_0]\) at which spike and smooth avert equally), the shift relative to the deterministic crossover \(D_0^*\) is

$$D_0^{*,\text{eff}} = D_0^* + \delta, \qquad \delta \approx \frac{\sigma^2}{2} \cdot \frac{\tilde S''_{\text{spike}}(D_0^*) - \tilde S''_{\text{smooth}}(D_0^*)}{\tilde f_{\text{spike}}(D_0^*) - \tilde f_{\text{smooth}}(D_0^*)}$$

and the sign of \(\delta\) equals the sign of the skewness:

  • Right-skewed (\(D_0^* > 0\)): \(\delta > 0\). Variance helps spikes. The crossover shifts right.
  • Left-skewed (\(D_0^* < 0\)): \(\delta < 0\). Variance hurts spikes. The crossover shifts left.
  • Symmetric (\(D_0^* = 0\)): \(\delta = 0\). Variance has no leading-order effect.

The mechanism is Jensen's inequality applied to the survival function's curvature. Past the mode (\(d > 0\)), every unimodal \(\tilde S\) is convex, and more concentrated profiles are more convex — so they benefit more from averaging over random \(D_0\). Before the mode (\(d < 0\)), the inequality reverses: \(\tilde S\) is concave, and concentration amplifies the penalty.

Since real infectiousness profiles are universally right-skewed (fast viral rise, slow immune-mediated decline), the practical consequence is unambiguous: noisy detection timing always favours punctuated transmission.
